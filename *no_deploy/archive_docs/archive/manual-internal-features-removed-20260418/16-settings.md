# 16. 店舗設定

管理ダッシュボードの「店舗設定」タブでは、店舗の各種設定を行います。マネージャー以上のロールで利用できます。

## 16.1 サブタブ一覧

| サブタブ | 説明 |
|---------|------|
| 営業設定 | 営業日区切り・税率・ラストオーダー等 |
| KDSステーション | KDSのステーション設定 |
| スタッフ管理 | 自店舗のスタッフアカウント管理 |
| 監査ログ | 操作履歴の閲覧 |
| 領収書設定 | レシートの表示内容設定 |

---

## 16.2 営業設定

### 営業基本設定

| 設定項目 | 初期値 | 説明 |
|---------|--------|------|
| 営業日区切り時刻 | 05:00 | この時刻でレポートの「1日」が区切られます。深夜営業の場合、翌日の5:00までが前日の売上として集計されます |
| 税率（%） | 10 | 店内飲食の標準消費税率 |
| 注文上限数 | 10 | セルフオーダーでの1回の注文の品目上限数 |
| 注文上限金額 | ¥30,000 | セルフオーダーでの1回の注文の金額上限 |
| ラストオーダー時刻 | — | 自動ラストオーダーの時刻。設定すると、この時刻にセルフオーダーの新規注文が自動ブロックされます |

### レジ設定

| 設定項目 | 初期値 | 説明 |
|---------|--------|------|
| レジ開け初期金額 | ¥30,000 | レジ開け時のデフォルトつり銭準備金 |
| 過不足閾値 | ¥500 | この金額を超える過不足がある場合、レジレポートで警告表示 |
| 現金 | ON | 現金決済の有効/無効 |
| クレジットカード | ON | カード決済の有効/無効 |
| QRコード | ON | QR決済の有効/無効 |

### レシート設定

| 設定項目 | 説明 |
|---------|------|
| 店舗名 | レシートに印刷する店舗名 |
| 住所 | レシートに印刷する住所 |
| 電話番号 | レシートに印刷する電話番号 |
| フッターテキスト | レシート下部に印刷するメッセージ（複数行対応） |

### Googleレビュー設定

| 設定項目 | 説明 |
|---------|------|
| Google Place ID | GoogleマップのプレイスID。設定すると、お客さんが高評価（4〜5つ星）を付けた際にGoogleレビューページへのリンクが表示されます |

### テイクアウト設定

テイクアウト機能の各設定については「9. テイクアウト管理」を参照してください。

### 外観カスタマイズ

| 設定項目 | 説明 |
|---------|------|
| テーマカラー | セルフオーダー画面のメインカラー。カラーピッカーまたはHEX値で指定。デフォルト：#ff6f00（セルフオーダー）、#2196F3（テイクアウト） |
| 表示名 | ヘッダーに表示するブランド名 |
| ロゴ画像URL | ヘッダーに表示するロゴ画像のURL。ライブプレビュー付き |

## 16.4 スタッフ管理

マネージャーは自店舗のスタッフアカウントを管理できます。

### スタッフの作成

| フィールド | 種類 | 必須 | 説明 |
|-----------|------|------|------|
| 表示名 | テキスト | ✅ | — |
| ユーザー名 | テキスト | ✅ | ログインID |
| パスワード | パスワード | ✅ | 初期パスワード |

### スタッフの編集

- 表示名の変更
- パスワードの変更
- 有効/無効の切替

::: warning マネージャーの制限
マネージャーが管理できるのは自分の所属店舗のスタッフのみです。他店舗のスタッフや他のマネージャーのアカウントは変更できません。他店舗のスタッフ管理はオーナーダッシュボードの「ユーザー管理」タブから行います。
:::

## 16.5 監査ログ

店舗内の全操作の履歴を閲覧できます。

### アクティブセッション管理

監査ログ画面では、現在アクティブなセッション（ログイン中の端末）も確認できます。

| 表示情報 | 説明 |
|---------|------|
| 端末情報 | ブラウザ/OS |
| IPアドレス | — |
| ログイン日時 | — |
| 最終操作日時 | — |

不審なセッションや退勤後のログアウト忘れに対して、「ログアウト」ボタンでリモートログアウトが可能です。

---

## 16.6 設定変更の反映タイミング

設定変更は保存後すぐに反映されます。ただし、以下の点に注意してください。

- **セルフオーダー画面：** お客さんが画面をリロードするか、メニューバージョンチェック（3分間隔）で更新されます
- **KDS画面：** 次のポーリング（3秒間隔）で反映されます
- **ハンディPOS：** 次のメニューバージョンチェック（3分間隔）で反映されます
- **レジ画面：** 次のポーリングで反映されます
