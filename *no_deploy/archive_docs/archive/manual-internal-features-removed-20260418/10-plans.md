# 10. プラン・コース管理

管理ダッシュボードの「プラン・コース」タブでは、食べ放題プラン（時間制限付きプラン）とコース料理テンプレートを管理します。マネージャー以上のロールで利用できます。

## 10.1 食べ放題プラン

### プラン一覧

「プラン・コース」タブ → 「食べ放題プラン」サブタブを選択すると、登録済みプランの一覧が表示されます。

各プランには以下の情報が表示されます。

| 表示要素 | 説明 |
|---------|------|
| プラン名 | 日本語名（英語名があればカッコ内に表示） |
| 制限時間 | 「90分」形式 |
| ラストオーダー | 「LO 15分前」形式 |
| 価格 | 「¥2,980」形式 |
| ステータスバッジ | 緑色「有効」/ 赤色「無効」 |
| 並び順 | 表示順の数値 |
| 「メニュー管理」ボタン | プラン専用メニューの展開 |
| 「編集」ボタン | プランの編集 |
| 「削除」ボタン | プランの削除 |

### プランの追加

1. 「プラン追加」ボタンをクリックします
2. モーダルが開きます
3. 以下のフィールドに入力します

| フィールド | 種類 | 必須 | 初期値 | 説明 |
|-----------|------|------|--------|------|
| プラン名 | テキスト | ✅ | — | 日本語名（例：「90分食べ放題」） |
| プラン名（英語） | テキスト | — | — | 英語名（例：「90min All-You-Can-Eat」） |
| 制限時間（分） | 数値 | ✅ | 90 | プランの時間制限（最小1分） |
| ラストオーダー（分前） | 数値 | — | 15 | 終了の何分前にラストオーダーとするか |
| 価格（円） | 数値 | — | 0 | プラン料金 |
| 並び順 | 数値 | — | 0 | 表示順（小さいほど上） |
| 説明 | テキストエリア（2行） | — | — | プランの説明文 |

4. 「保存」ボタンをクリックします

### プランの編集

既存プランの編集では、上記フィールドに加えて以下が変更可能です。

| フィールド | 種類 | 説明 |
|-----------|------|------|
| 有効/無効 | セレクト | 無効にすると着席時のプラン選択に表示されなくなります |

### プラン専用メニュー

食べ放題プランには専用のメニュー品目を登録できます。「メニュー管理」ボタンをクリックすると、プランに紐づく品目の一覧が展開されます。

**品目の追加：**

1. 「+ 品目追加」ボタンをクリックします
2. モーダルが開きます
3. 以下のフィールドに入力します

| フィールド | 種類 | 必須 | 説明 |
|-----------|------|------|------|
| 品名 | テキスト | ✅ | 品目の日本語名 |
| 品名（英語） | テキスト | — | 品目の英語名 |
| カテゴリ | セレクト | — | 分類カテゴリ |
| 説明 | テキストエリア（2行） | — | 品目の説明 |
| 並び順 | 数値 | — | 表示順 |

4. 「保存」ボタンをクリックします

各品目は編集・削除が可能です。無効にすることもできます。

## 10.2 コース料理

### コース一覧

「プラン・コース」タブ → 「コース管理」サブタブを選択すると、登録済みコースの一覧が表示されます。

各コースには以下の情報が表示されます。

| 表示要素 | 説明 |
|---------|------|
| コース名 | 日本語名（英語名があればカッコ内に表示） |
| フェーズ数 | 「3フェーズ」形式 |
| 価格 | 「¥5,000」形式 |
| ステータスバッジ | 有効/無効 |
| 説明 | コースの説明文 |

### コースの追加

1. 「コース追加」ボタンをクリックします
2. 以下のフィールドに入力します

| フィールド | 種類 | 必須 | 初期値 | 説明 |
|-----------|------|------|--------|------|
| コース名 | テキスト | ✅ | — | 日本語名（例：「松コース」） |
| コース名（英語） | テキスト | — | — | 英語名 |
| 価格（円） | 数値 | — | 0 | コース料金 |
| 並び順 | 数値 | — | 0 | 表示順 |
| 説明 | テキストエリア（2行） | — | — | コースの説明文 |

3. 「保存」ボタンをクリックします

### フェーズの管理

コース料理はフェーズ（段階）で構成されます。各フェーズは順番に提供される料理のグループです。

**例：松コース（3フェーズ）**
1. フェーズ1「前菜」：サラダ ×1、スープ ×1
2. フェーズ2「メイン」：刺身盛り合わせ ×1、天ぷら ×1
3. フェーズ3「デザート」：抹茶アイス ×1、コーヒー ×1

### フェーズの追加

1. コースの「+ フェーズ追加」ボタンをクリックします
2. 以下のフィールドに入力します

| フィールド | 種類 | 必須 | 説明 |
|-----------|------|------|------|
| フェーズ名 | テキスト | ✅ | フェーズの名前（例：「前菜」）。プレースホルダー：「例: 前菜」 |
| フェーズ名（英語） | テキスト | — | 英語名 |
| 自動発火（分後） | 数値 | — | 前のフェーズからN分後に自動で次のフェーズを開始。空欄の場合は手動で進める |
| 品目 | テキストエリア（4行） | ✅ | 1行に1品目。形式：「品名」または「品名 ×数量」 |

**品目の入力形式：**
```
サラダ
スープ ×2
刺身盛り合わせ ×1
```

- 数量を省略した場合は1として扱われます
- `×`（全角）と`x`（半角）のどちらも使用できます

3. 「保存」ボタンをクリックします
4. フェーズ番号は自動的に連番で付与されます

### フェーズの発火方式

| 方式 | 表示 | 動作 |
|------|------|------|
| 手動 | 「手動」 | KDS画面の「次フェーズ」ボタンで進める |
| 自動 | 「自動 N分後」 | 前のフェーズ開始からN分後に自動で次のフェーズの料理がKDSに表示される |

### コースの運用フロー

1. **コースを作成し、フェーズと品目を登録する**（事前準備）
2. **お客さんの着席時にコースを選択する**（ハンディPOSのテーブル状況タブ → 着席モーダル）
3. **フェーズ1の料理が自動的にKDSに表示される**
4. **キッチンスタッフが調理・提供する**
5. **KDS画面の「次フェーズ」ボタンで次のフェーズに進める**（手動の場合）
6. **全フェーズが完了するまで繰り返す**

::: warning コース削除時の注意
コースを削除すると、そのコースに紐づく全てのフェーズも削除されます（カスケード削除）。
:::
