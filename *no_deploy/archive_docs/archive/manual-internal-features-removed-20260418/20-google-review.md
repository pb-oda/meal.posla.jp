# 20. Googleレビュー連携と満足度評価

POSLAには、お客さんの満足度を収集し、高評価のお客さんをGoogleレビューに誘導する仕組みがあります。

## 20.1 満足度評価の仕組み

### 評価のタイミング

お客さんのセルフオーダー画面で、注文品目が全て「served（提供済）」になると、満足度評価のプロンプトが表示されます。

### 評価の方法

星（またはフェイス）ベースの評価インターフェースが表示されます。お客さんは直感的に満足度を入力できます。

### 評価データの活用

- スタッフ評価レポートで平均評価として集計されます
- 個別の注文に紐づいて記録されます

---

## 20.2 Googleレビューへの誘導

### 設定方法

1. 管理ダッシュボードの「店舗設定」→「営業設定」を開きます
2. 「Googleレビュー設定」セクションのGoogle Place IDを入力します

**Google Place IDの取得方法：**
1. [Google Maps](https://www.google.com/maps) で自店舗を検索します
2. 店舗の詳細ページのURLからPlace IDを取得します
3. または [Place ID Finder](https://developers.google.com/maps/documentation/places/web-service/place-id) を使用します

### 誘導の流れ

1. お客さんが満足度評価で**4つ星以上（高評価）**を付けます
2. 「ありがとうございます！Googleでレビューを書いていただけませんか？」というメッセージとリンクが表示されます
3. リンクをタップするとGoogleマップのレビュー投稿画面が開きます

**3つ星以下（低評価）の場合：**
Googleレビューへのリンクは表示されません。これにより、低評価のレビューがGoogleに直接投稿されるリスクを軽減します。

::: tip Googleレビューの重要性
Googleマップでの高評価とレビュー数は、新規顧客の集客に大きく影響します。満足したお客さんを自然にレビュー投稿へ誘導することで、店舗の評価向上が期待できます。
:::
