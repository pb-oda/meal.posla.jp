13. レポート ​

管理ダッシュボードの「レポート」タブでは、売上・レジ・回転率・スタッフ評価・併売分析のレポートを閲覧できます。マネージャー以上のロールで利用できます。

基本レポート（売上レポート）はstandardプランで利用でき、高度レポート（回転率・スタッフ評価・併売分析）はproプラン以上で利用できます。

13.1 共通操作 ​

期間の選択 ​

全てのレポートで期間を指定できます。プリセットボタンとカスタム範囲が用意されています。

プリセット期間
今日本日の営業日
昨日前日の営業日
今週今週（月曜〜日曜）
今月今月（1日〜末日）

カスタム範囲では開始日と終了日を自由に指定できます。

CSVエクスポート ​

多くのレポートでCSVエクスポートに対応しています。データを外部ツール（Excel等）で分析する際に利用します。

13.6 併売分析（proプラン以上） ​

「レポート」タブ → 「併売分析」サブタブ

アソシエーションルール ​

一緒に注文される品目の組み合わせを分析します（マーケットバスケット分析）。

最小サポート閾値の設定： 2 / 3 / 5 / 10 から選択（最低何回以上の同時注文があれば表示するか）

列説明
品目A—
品目B—
サポート全注文中での同時出現回数
信頼度（A→B）Aを注文した人がBも注文する確率
信頼度（B→A）Bを注文した人がAも注文する確率
リフト値相関の強さ

リフト値の解釈：

  • 1.5以上（緑色）： 強い正の相関（一緒に注文されやすい）
  • 1.0前後： 無相関
  • 1.0未満（赤色）： 負の相関（一緒に注文されにくい）

品目頻度ランキング ​

注文頻度の上位20品目をバーチャートで表示します。

併売分析の活用

  • リフト値の高い組み合わせをセットメニューとして提案
  • 「一緒にいかがですか？」のレコメンド根拠に活用
  • メニュー配置（カテゴリ順やおすすめ表示）の最適化に活用
