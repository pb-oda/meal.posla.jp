カテゴリ一覧

メニューの大分類。ここでの並び順がセルフメニュー画面のカテゴリタブ順に反映されます。

オプショングループ

トッピングやサイズ違いなど、メニュー品目に付けられる選択肢のセット。1 つ作っておくと複数品目で再利用できます。

店舗メニュー

店舗で実際に提供しているメニューの一覧と価格・売切れ設定。本部マスタと連動し、店舗独自の上書き（価格 / 非表示 / 売切れ）が可能です。多言語翻訳と CSV で一括管理もできます。

今日のおすすめ設定

店舗限定メニュー

本部マスタに紐づかない、この店舗だけの限定品。本部配信の影響を受けず、店舗の裁量で登録できます。季節商品や日替わりに便利。

テーブル一覧

店内の座席。各テーブルに QR が 1 枚割り当てられ、お客さんがスマホで読み取ると自動でそのテーブルの注文画面が開きます。「個別QR」は相席時の席ごと会計用。

フロアマップ

店内の座席レイアウトを図で管理。テーブル位置をドラッグで配置し、ホールスタッフの視覚的把握と予約台帳との連動に使います。

売上レポート

日次 / 時間帯別の売上と客数。閉店後の振り返りと翌日の仕込み判断に使います。

レジ分析

レジ締め時の過不足、決済手段別の内訳、割引の適用状況。日次のレジ管理と不正検知に使います。

注文履歴

過去の注文データの検索。顧客対応時の「このとき何を頼みましたか？」の確認や、クレーム調査に使います。

回転率・調理時間・客層分析

テーブルの回転時間、オーダーから提供までの調理時間、客層（人数 / 時間帯）の分析。ピークの負荷設計と仕込みに使います。

スタッフ別評価・監査

スタッフごとの注文数、客単価、打刻精度などを集計。シフト調整や評価の参考資料に使います。

満足度・低評価分析

お客さまの評価の時系列と、低評価のコメント抽出。サービス改善の優先順位付けに使います。

併売分析（バスケット分析）

一緒に注文されやすい品目の組み合わせ。おすすめ提案やセット化の判断材料に使います。

食べ放題プラン

食べ放題（90 分 / 飲み放題 等）の条件と対象メニュー。お客さま側のプラン選択画面とセルフオーダーに連動します。

コース料理

コース（前菜 → 主菜 → デザート の順で提供）の登録と所要時間。予約画面・予約台帳のコース選択で使われます。

在庫・レシピ管理

原材料の在庫数とレシピ（メニュー品目 → 原材料使用量）を管理します。レシピを登録しておくと、注文ごとに自動で在庫が引き落とされ、アラートや売切れ判定に反映されます。

店舗設定

営業時間、ブランディング、税率、通知先メールなど店舗の基本設定。変更は全体に影響するため、慎重に。

KDSステーション

キッチンディスプレイの割り振り（調理場別・メニュー別）。メニューカテゴリと KDS 端末を結び付けます。

スタッフ管理

人のスタッフアカウント（オーナー / マネージャー / スタッフ）の作成・編集。シフト・勤怠・人件費の対象になります。1 人 1 アカウントが原則です。

デバイスアカウント（KDS / レジ端末）

スタッフではなく端末専用のログインアカウント。1 端末 = 1 アカウントで、シフト・人件費・スタッフレポート・監査ログのスタッフ集計から除外されます。

監査ログ

誰がいつどの操作をしたかの記録。トラブル調査や内部統制のために参照します。保持期間は 1 年。

エラー監視

サーバー側で記録された API エラーの件数と発生源。障害の早期発見と改善優先度の判断に使います。

テイクアウト注文管理

お客さま側から入ったテイクアウト注文の一覧と調理・準備状況。受取時刻順で流れを管理します。

AIアシスタント

Gemini を使った業務アシスタント。SNS 投稿文案、メニュー分析、需要予測などを自然言語で依頼できます。

領収書・インボイス

領収書や適格請求書の印字設定（登録番号、発行条件、但し書き）。インボイス制度対応のため、あらかじめ設定しておきます。

週間カレンダー

週間のシフト割当を一覧で確認・編集。行はスタッフ、列は日付。セルタップで時間編集ダイアログが開きます。AI 提案は別ボタンから呼び出し可能。

勤怠一覧

スタッフの出退勤打刻の一覧。打刻忘れや時刻誤りは各行の「修正」から編集できます（監査ログに記録）。

テンプレート

曜日 × 時間帯で固定シフトを事前登録。テンプレートから週次のシフト自動生成が可能で、週のベース組み立てに使います。

シフト設定

シフト管理の運用パラメータ。希望提出の締切日・デフォルト休憩・残業閾値・GPS 打刻の必須化・スタッフ画面に表示するツールを店舗単位で設定します。

集計サマリー

週次・月次の総労働時間と人件費。労基法違反の自動チェック（週40時間超・連続勤務等）付き。閉店後の振り返りに使います。

ヘルプ

店舗間のヘルプ要請。人手不足時に他店舗へ応援依頼を送信、受信側は承認/却下で対応。
