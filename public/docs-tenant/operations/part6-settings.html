Part 6 — 店舗の設定 ​

このページでは、店舗運営に必要な各種設定を、実際に押すボタン 1 つずつ順番に説明します。

このページの範囲

本ページで触る画面は以下の 3 つです。

  • オーナー管理画面: 店舗管理・ユーザー管理・決済設定・本部メニュー
  • 管理画面（マネージャー以上）: 店舗設定タブ内の各種設定
  • 店舗設定サブタブ: 営業時間・レジ・税率・ウェルカムメッセージ・プリンタ・外観カスタマイズ

6-1. 店舗情報の変更 ​

6-1-1. 店舗名を変更する（オーナーのみ） ​

  1. オーナーアカウントでオーナー管理画面を開く
  2. 上部タブから 【店舗管理】 をクリック
  3. 店舗一覧が表示される
  4. 該当店舗の行にある 【編集】 ボタン
  5. モーダルが開く
  6. 店舗名 欄を書き換え
  7. 【保存】
  8. 数秒で反映（管理画面のヘッダーにも新しい名前が表示）

6-1-2. 住所・電話番号を登録する（レシート印字用） ​

お店のレシートに印字される住所・電話番号は別の画面で設定します。

  1. マネージャー / オーナーで管理画面にログイン
  2. 上部タブの 【店舗設定】 をクリック
  3. サブタブから 【店舗設定】 を選択
  4. 画面を下にスクロールして 「レシート設定」 セクションを探す
  5. 以下を入力:
    • 店舗名（レシートに印字される名前。オーナー管理画面の店舗名とは別に設定できます）
    • 住所（例: 「東京都渋谷区渋谷 1-2-3 XX ビル 5F」）
    • 電話番号（例: 「03-1234-5678」）
  6. 【設定を保存】 ボタン

レシート用の店舗名と、システム上の店舗名は別

システム上の「店舗名」は管理画面のヘッダーや店舗選択に使われる内部識別用。レシート印字用の店舗名は、お客様に見える形式（例: 「松乃家 渋谷店」）を設定できます。

6-1-3. 新しい店舗を追加する（オーナーのみ） ​

多店舗経営で新店舗をオープンする場合。

  1. オーナー管理画面 → 【店舗管理】
  2. 【+ 店舗を追加】 ボタン
  3. モーダル:
    • 店舗名
    • 店舗 Slug（URL 用の識別子、半角英小文字とハイフン）
    • 英語名（任意）
  4. 【保存】
  5. 次回請求から追加店舗分の料金が発生

店舗追加の料金

  • 2 店舗目以降は ¥17,000 / 月 / 店舗（1 店舗目は ¥20,000）
  • 次回請求日から反映（日割り計算されるケースあり）
  • 本部一括メニュー配信アドオン契約中の場合、追加店舗分も +¥3,000 / 月 / 店舗 上乗せ

6-2. テーブルの追加・削除・レイアウト変更 ​

6-2-1. テーブルを追加する ​

  1. 管理画面 → 上部タブ 【テーブル・フロア】 → サブタブ 【テーブル管理】
  2. 【+ テーブル追加】 ボタン
  3. モーダル:
    • テーブルコード（例: T06、店内で一意）
    • 座席数（例: 4）
    • フロア（例: 1F、テラス）
  4. 【保存】
  5. 一覧に追加される

6-2-2. テーブルを削除する（推奨: 無効化） ​

過去の注文履歴との紐付けを壊さないため、完全削除ではなく「無効化」 を推奨。

  1. 該当テーブルの 【編集】 ボタン
  2. 「有効」トグル を OFF
  3. 【保存】
  4. 一覧で「無効化」バッジ表示
  5. セルフメニュー・ハンディから見えなくなる

6-2-3. 完全削除（非推奨） ​

どうしても完全削除したい場合:

  1. 該当テーブル → 【削除】 ボタン（赤）
  2. 確認ダイアログで 【OK】
  3. 削除される（注意: 過去の注文との紐付けが分かりにくくなる場合があります）

6-2-4. フロアマップでのレイアウト変更 ​

  1. サブタブ 【フロアマップ】
  2. 画面にテーブルのカード一覧が表示される
  3. 各テーブルの位置座標（pos_x / pos_y）・サイズ（width / height）を編集
  4. 【保存】

ドラッグ＆ドロップ対応は今後実装予定

現在のフロアマップはカード一覧形式。ドラッグ&ドロップによるビジュアル配置機能は今後のアップデートで。

6-3. QR コードの印刷 ​

6-3-1. テーブル QR の印刷 ​

  1. 管理画面 → 【テーブル・フロア】 → 【テーブル管理】
  2. 各テーブルの行にある 【QR 印刷】 ボタン
  3. 新しいタブで印刷用ページが開く
  4. Cmd+P（Mac）または Ctrl+P（Windows）で印刷
  5. テーブルごとに 1 枚ずつ印刷される

6-3-2. テイクアウト QR の印刷 ​

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 画面下部の 「テイクアウト QR」セクション
  3. 【🖨 印刷】 ボタン → 印刷

6-3-3. 予約 LP QR の印刷 ​

  1. 同じく【店舗設定】 → 【店舗設定】サブタブ
  2. 下部の 「📅 予約LP」セクション
  3. 【🖨 印刷】 ボタン → A4 ポスターとして印刷

6-4. 営業時間・消費税率・注文上限の設定 ​

6-4-1. 設定項目 ​

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 「営業基本設定」セクション に以下の項目:
項目推奨値説明
営業日区切り時刻通常 05:00深夜営業店の売上集計日を決める時刻。24:00 を過ぎても当日扱い
税率（店内）10%標準税率
税率（テイクアウト）8%軽減税率
注文上限数10セルフオーダーの 1 注文あたり品目数上限
注文上限金額¥30,000セルフオーダーの 1 注文あたり金額上限
ラストオーダー時刻閉店 30 分前以降セルフ注文がブロックされる

6-4-2. 消費税率の注意 ​

  • 日本の消費税は 店内 10% / テイクアウト軽減税率 8% が標準
  • 輸出店舗・免税店は別設定が必要（運営にご相談ください）

6-5. 決済方法の有効化・無効化 ​

お店で使う支払方法（現金・カード・QR 決済）を個別に ON/OFF できます。

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 「レジ設定」セクション に以下のトグル:
方法デフォルト用途
現金ON現金決済を受け付けるか
クレジットカードON店舗契約の外部カード端末で決済済みとして記録
QRコードONPayPay / d払い等、店舗契約QRで決済済みとして記録
  1. 【設定を保存】

6-6. POSLA決済の設定（セルフ会計 / テイクアウト） ​

詳細は 21 章「決済設定」 参照。ここでは要約のみ。

手順 ​

  1. オーナーアカウントでオーナー管理画面にログイン
  2. 上部タブから 【決済設定】
  3. Stripe Connect のオンボーディングを開始
  4. Stripe画面で必要情報を入力
  5. POSLAに戻り、Connect有効状態を確認
  6. セルフ会計またはテイクアウトをONにする

通常レジのカード/QRは、この設定とは別です。店舗が契約している決済端末・QR決済アプリで支払いを完了し、POSLAには記録だけ残します。

6-7. ウェルカムメッセージの設定 ​

お客様がセルフメニューを開いた時の挨拶文。

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 「ウェルカムメッセージ」セクション
  3. 日本語版 を入力（例: 「いらっしゃいませ！今日のおすすめは唐揚げ定食です」）
  4. 英語版 を入力（任意、例: Welcome! Today's recommendation is Karaage Teishoku.）
  5. 【設定を保存】

6-8. テイクアウトの設定 ​

テイクアウト受付の各種設定。

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 「テイクアウト設定」セクション:
項目説明
テイクアウト受付トグル ON/OFF
受付開始時刻例: 10:00
受付終了時刻例: 20:00
スロット容量1 時間あたりの同時受付数（例: 5 件）
最低準備時間注文から受取までの最短時間（例: 15 分）
オンライン決済必須化2026-04-18 以降デフォルト ON、店頭支払いは廃止
  1. 【設定を保存】

6-9. プリンタ（Star / Epson）の設定（U-2 新機能） ​

詳細は 26 章 参照。ここでは要約のみ。

6-9-1. プリンタを Wi-Fi に接続する ​

  1. プリンタの電源を入れる
  2. プリンタ本体の FEED ボタン を長押し（約 5 秒）
  3. 自動印字される紙に IP アドレス が書いてある（例: 192.168.1.234）
  4. 紙をメモしておく

6-9-2. POSLA にプリンタを登録 ​

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 画面を下にスクロール
  3. 「プリンタ設定」セクション
  4. 以下を入力:
項目値説明
プリンタ種別Star / Epson / ブラウザメーカーを選択
IP アドレス192.168.1.234上記でメモした値
ポート80通常は 80 でOK
用紙幅58mm / 80mmプリンタの仕様
自動印刷（キッチン伝票）ON注文時に厨房に伝票を印刷
自動印刷（レシート）ON会計時にレシートを印刷
  1. 【🖨 テスト印刷】 ボタン → テスト用紙が出る
  2. 【設定を保存】

6-10. 外観カスタマイズ（ロゴ・色） ​

ブランドの世界観を反映。

  1. 管理画面 → 【店舗設定】 → 【店舗設定】サブタブ
  2. 「外観カスタマイズ」セクション:
項目説明
テーマカラーカラーピッカーまたは HEX 値（例: #ff6f00）でブランドカラー指定
表示名ヘッダー等に表示される店舗名（多言語対応）
ロゴ画像URLロゴ画像の URL（HTTPS 必須）
  1. 【設定を保存】

ロゴ画像の準備 ​

  • 画像は事前に自社ホームページや画像管理サービスへアップロード
  • URL は HTTPS でなければならない（セキュリティ上の理由）
  • 推奨サイズ: 400×100px 以下、PNG 透過推奨

6-11. スタッフ管理（復習） ​

詳細は Part 2 参照。店舗設定タブから開けます。

  1. 管理画面 → 【店舗設定】 → サブタブ 【スタッフ管理】
  2. スタッフ一覧・デバイス一覧
  3. 【+ 追加】 でスタッフ新規作成
  4. 【+ デバイス追加】 でデバイスアカウント新規作成
  5. 【トークン発行】 でデバイス自動ログイン用ワンタイムトークン発行

6-12. KDS ステーション設定 ​

大規模キッチンで、料理ジャンルごとに KDS 画面を分ける設定。

  1. 管理画面 → 【店舗設定】 → サブタブ 【KDSステーション】
  2. 【+ ステーション追加】 ボタン
  3. 入力:
    • ステーション名（例: 「和食ステーション」）
    • 英語名（オプション）
    • 担当カテゴリ（チェックボックスで選択）
  4. 【保存】
  5. KDS 画面上部のステーション選択バーに追加される

詳細は 7 章「KDS」参照。

6-13. 監査ログの閲覧 ​

誰が何をしたか履歴を確認できる機能（manager 以上）。

  1. 管理画面 → 【店舗設定】 → サブタブ 【監査ログ】
  2. 日付範囲・ユーザー・イベントタイプでフィルタ可能
  3. 表示される情報:
    • 日時
    • ユーザー名・ロール
    • 操作内容（ログイン / メニュー編集 / 売り切れ切替 等）
    • 対象リソース

監査ログは、設定変更・削除・ログインなど 重要操作 を中心に記録します。

6-14. よくある質問 ​

Q1. 店舗を削除したい

A1. オーナー管理画面の店舗管理で 無効化 できます。過去データは保持されます。

Q2. 税率を変更したら既存の注文はどうなる？

A2. 過去の注文には影響しません。変更後の新規注文から新税率が適用されます。

Q3. プリンタが反応しない

A3. 26 章のトラブルシューティング参照。主な原因: Wi-Fi 接続切れ / IP アドレス変更 / ファームウェア古い。

Q4. 営業時間外でも管理画面は使える？

A4. 使えます。管理画面の操作は 24 時間いつでも可能。営業時間はお客様のセルフオーダーの受付時間に関する設定です。
