POSLA 店舗運用手順書 — 全体目次 ​

対象読者: 飲食店オーナー / マネージャー / アルバイトスタッフ (ITリテラシー不問)

書き方の方針:

  • 「設定タブを開く」ではなく 「画面上部の左から4番目にある ⚙ マーク (歯車のアイコン) の【設定】ボタンをタップします」 のように1クリック単位で記述
  • 専門用語は初出時に必ず説明
  • 失敗したときの対処法を必ず併記
  • 初めて使う方でも、上から順番に読めば店舗運用の流れが分かる構成にしています

全体構成 ​

Part 0: はじめに — POSLA を使う前の準備 ​

  • 0-1. 必要な機材一覧 (タブレット / スマホ / 店舗決済端末 / Wi-Fiルーター / プリンタ)
  • 0-2. Wi-Fi の確認方法 (iPhone / Android / Windows それぞれ)
  • 0-3. ブラウザの開き方と URL の入力方法
  • 0-4. ログイン画面の開き方
  • 0-5. ログインの仕方 (ユーザー名とパスワードの入力)
  • 0-6. パスワードを忘れたときの対処法
  • 0-7. 画面の見方 (タブ・ボタン・アイコンの名前と場所)
  • 0-8. ロール (権限) の違い: オーナー / マネージャー / スタッフ / デバイス

Part 1: 毎日の営業 (開店〜閉店) ​

  • 1-1. 開店準備
    • 1-1-1. KDS の電源を入れてログインする
    • 1-1-2. レジの電源を入れてログインする
    • 1-1-3. ハンディ端末の準備
    • 1-1-4. 店頭決済端末の接続確認
    • 1-1-5. プリンタの接続確認
    • 1-1-6. レジを開ける (レジ開け操作の全手順)
  • 1-2. お客さんが来たとき
    • 1-2-1. テーブルに案内する (空席確認)
    • 1-2-2. テーブル開放認証 (📲 QR開放ボタンの押し方)
    • 1-2-3. お客さんが QR コードを読み込む流れの説明
    • 1-2-4. ハンディで注文を入れる (スタッフ側操作)
  • 1-3. キッチンでの操作
    • 1-3-1. KDS 画面の見方 (受付 / 調理中 / 提供待ち)
    • 1-3-2. 注文を「調理開始」にする
    • 1-3-3. 調理が終わったら「提供待ち」にする
    • 1-3-4. 音声コマンドで操作する方法 (「○○番提供」)
  • 1-4. 料理を提供する
    • 1-4-1. KDS で「提供済み」にする
    • 1-4-2. テーブルの状態を確認する
  • 1-5. お会計
    • 1-5-1. 現金でのお会計手順 (おつりの計算)
    • 1-5-2. カードでのお会計手順 (外部カード端末で決済後、POSLAに記録)
    • 1-5-3. QR決済 (PayPay/d払い) でのお会計手順
    • 1-5-4. セルフレジ (お客さん自身のスマホ決済)
    • 1-5-5. レシートの発行方法
    • 1-5-6. 個別会計 (割り勘) の方法
  • 1-6. テイクアウト注文の処理
  • 1-7. 品切れの設定と解除
  • 1-8. ラストオーダーの設定
  • 1-9. 閉店作業
    • 1-9-1. レジを閉める (レジ締め操作の全手順)
    • 1-9-2. 売上の確認方法
    • 1-9-3. KDS / ハンディ / POSレジ端末の閉店時確認

読み進め方 ​

初めてPOSLAを使う場合は、Part 0 から順番に読んでください。すでに運用中の場合は、困っている操作に近いPartを開いて確認してください。

店舗スタッフに共有する場合は、まず Part 1「毎日の営業」と Part 8「トラブルシューティング」を案内すると、日々の営業で必要な操作を把握しやすくなります。
