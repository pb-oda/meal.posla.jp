3. 管理ダッシュボード ​

管理ダッシュボード（dashboard.html）は、マネージャーとスタッフが日常的に使用するメイン画面です。タブで機能を切り替えながら、店舗運営に必要な全ての操作を行います。

3.1 画面構成 ​

ヘッダー ​

画面上部のヘッダーには以下の要素があります。

位置要素説明
左テナント名あなたの企業/ブランド名が表示されます
左店舗セレクタードロップダウンで操作対象の店舗を切り替えます。複数店舗に所属している場合に使用します。選択した店舗はブラウザに記憶され、次回アクセス時も同じ店舗が選択されます
右ショートカットリンク「ハンディ」「KDS」「レジ」「注文画面」への直接リンク。新しいタブで開きます
右ユーザー名現在ログイン中のユーザー名が表示されます
右ログアウトボタンクリックでログアウトします

メインタブ（グループタブ） ​

ヘッダーの下に、横並びのメインタブが表示されます。タブをクリックすると、対応する機能グループが表示されます。

タブ名対象ロール内容
メニュー管理マネージャー以上カテゴリ・メニュー・オプション・限定メニューの管理
プラン・コースマネージャー以上食べ放題プラン・コース料理の管理
テーブル・フロアマネージャー以上テーブル登録・フロアマップの配置
在庫・レシピマネージャーのみ食材・レシピ・棚卸しの管理（proプラン以上）
レポートマネージャー以上売上・レジ・回転率・スタッフ評価・併売分析
店舗設定マネージャー以上営業設定・KDSステーション・スタッフ管理・監査ログ・領収書設定
テイクアウトマネージャー以上テイクアウト注文の受付・管理（proプラン以上）
AIアシスタントマネージャー以上SNS投稿生成・売上分析・競合調査・需要予測
シフト管理スタッフ以上シフト管理・勤怠連携（proプラン以上）

スタッフが見えるタブ

スタッフロールでログインした場合、「シフト管理」タブのみが表示されます。その中でも「マイシフト」「希望提出」「勤怠打刻」のみ操作可能で、マネージャー向けのサブタブ（カレンダー、テンプレート、設定、サマリー等）は非表示です。

サブタブ ​

メインタブをクリックすると、その中にサブタブが表示されます。サブタブで具体的な機能を切り替えます。

例：「メニュー管理」タブのサブタブ

サブタブ名説明
カテゴリ管理メニューのカテゴリ（前菜、メイン、ドリンク等）を作成・編集
オプション管理トッピング・サイズ等のオプショングループと選択肢を管理
店舗メニュー本部メニューの店舗別カスタマイズ（価格上書き・非表示・品切れ）
限定メニューその店舗だけのオリジナルメニューを追加

例：「シフト管理」タブのサブタブ

サブタブ名対象ロール説明
週間カレンダーマネージャーシフトの割当・編集を行うメイン画面
マイシフトスタッフ自分の確定シフトを閲覧
希望提出スタッフ希望シフトの日時を提出
勤怠一覧マネージャー全スタッフの出退勤記録一覧
テンプレートマネージャーシフトパターン（早番・遅番等）の定義
シフト設定マネージャー提出締切日・休憩デフォルト・GPS等の設定
集計サマリーマネージャー労働時間・人件費・労基法警告の集計
ヘルプマネージャー他店舗へのスタッフ派遣要請（enterpriseプラン）

コンテンツエリア ​

サブタブの下に、選択中の機能のコンテンツが表示されます。ここに一覧表・フォーム・グラフ等が描画されます。

勤怠打刻ボタン（スタッフ専用） ​

スタッフロールでログインすると、画面のヘッダー付近に出勤ボタンと退勤ボタンが常時表示されます。これは全てのタブで表示され、いつでも打刻できます。マネージャー・オーナーには表示されません。

タブの操作 ​

タブの切り替え ​

  1. メインタブ（横並び）をクリックすると、対応するサブタブ群が表示されます
  2. サブタブをクリックすると、コンテンツエリアの内容が切り替わります
  3. データはタブ切り替え時に自動で読み込まれます。手動リフレッシュは不要です

店舗の切り替え ​

複数店舗に所属している場合：

  1. ヘッダーの店舗セレクター（ドロップダウン）をクリック
  2. 操作したい店舗を選択
  3. 表示中のデータが選択した店舗のものに自動的に切り替わります
  4. 選択した店舗はブラウザに保存され、次回ログイン時も同じ店舗が選ばれます

注意

店舗を切り替えると、現在表示中のデータ（メニュー、注文、シフト等）は全て切り替え先の店舗のものに変わります。編集中の内容がある場合は、保存してから切り替えてください。

別画面へのショートカット ​

ヘッダー右側のショートカットリンクから、以下の画面を新しいタブで直接開けます。

リンク名遷移先用途
ハンディ/handy/ハンディPOS画面（注文入力端末）
KDS/kds/キッチンディスプレイ画面
レジ/kds/cashier.htmlPOSレジ画面
注文画面/customer/menu.htmlお客さん向けセルフオーダー画面（確認用）

これらのリンクは新しいタブで開きます

管理ダッシュボードを閉じずに、別の画面を同時に操作できます。例えば、ダッシュボードでメニューを編集しながら、別タブでセルフオーダー画面を開いて表示を確認する、といった使い方ができます。

プラン外の機能 ​

契約中のプラン（standard/pro/enterprise）に含まれない機能のタブは自動的に非表示になります。例えば、standardプランではシフト管理タブや在庫管理タブは表示されません。プランのアップグレードについてはオーナーダッシュボードの「契約・プラン」タブから手続きできます。
