4. メニュー管理 ​

メニュー管理では、お客さんに表示するメニューの作成・編集・品切れ管理を行います。POSLAのメニューは「本部テンプレート → 店舗オーバーライド」の2層構造になっており、本部で統一メニューを管理しつつ、店舗ごとに価格や表示をカスタマイズできます。

管理ダッシュボードの**「メニュー管理」タブ**をクリックし、サブタブを選択して操作します。

4.1 カテゴリ管理 ​

カテゴリは、メニューをグループ分けする単位です。セルフオーダー画面やハンディPOS画面でカテゴリごとにメニューが表示されます。

サブタブ：「カテゴリ管理」

カテゴリの作成 ​

  1. 「カテゴリ管理」サブタブを開く
  2. 「カテゴリ追加」ボタンを押す
  3. モーダルが開く
  4. 以下を入力する：
項目必須説明入力例
カテゴリ名✓日本語の表示名前菜
カテゴリ名（英語）—英語の表示名。多言語対応時に使用Appetizers
  1. 保存ボタンを押す
  2. カテゴリ一覧に追加される

カテゴリの編集 ​

  1. カテゴリ一覧で、編集したいカテゴリの編集ボタン（鉛筆アイコン）を押す
  2. モーダルが開き、現在の値が表示される
  3. 以下を変更できる：
    • カテゴリ名
    • カテゴリ名（英語）
    • 表示順序（数値。小さいほど上に表示）
  4. 保存ボタンを押す

カテゴリの削除 ​

  1. 削除したいカテゴリの削除ボタン（ゴミ箱アイコン）を押す
  2. 確認ダイアログが表示される
  3. OKを押すと削除される

注意

カテゴリを削除すると、そのカテゴリに属するメニューは「カテゴリなし」の状態になります。削除前に、メニューを別のカテゴリに移動させることを推奨します。

カテゴリの並び順 ​

カテゴリの表示順序は「表示順序」の数値で決まります。数値が小さいカテゴリが上（左）に表示されます。セルフオーダー画面・ハンディPOS画面でも同じ順序で表示されます。

メニューテンプレート ​

メニューテンプレートは、全店舗で共通のマスターメニューです。ここで作成したメニューが全店舗の基本メニューになります。店舗ごとの価格変更や非表示設定は「店舗メニューオーバーライド」で行います。

サブタブ：「カテゴリ管理」（カテゴリ一覧とメニューが同じ画面に表示されます）

メニューの作成 ​

  1. 「メニュー追加」ボタンを押す
  2. モーダルが開く
  3. 以下を入力する：
項目必須説明入力例
メニュー名✓日本語の表示名特製唐揚げ
メニュー名（英語）—英語の表示名Special Karaage
カテゴリ✓ドロップダウンからカテゴリを選択メイン料理
価格（税込）✓税込価格（円）。0円も可880
説明文—メニューの詳細説明自家製タレで味付けした鶏もも肉...
説明文（英語）—英語の説明文Chicken thigh marinated in...
画像—メニュー写真。ドラッグ＆ドロップまたはクリックでアップロード—
  1. 保存ボタンを押す
  2. メニュー一覧に追加される

画像について

  • 対応形式：JPEG、PNG、WebP、GIF
  • 最大ファイルサイズ：5MB
  • アップロード時に自動的に800×800ピクセル以下にリサイズされます（品質0.8のJPEG）
  • 画像なしでも作成できます。画像なしの場合は🍽️アイコンが表示されます

メニューの編集 ​

  1. メニュー一覧で、編集したいメニューの編集ボタンを押す
  2. モーダルが開き、現在の値が表示される
  3. 各項目を変更する
  4. 品切れチェックボックス：チェックを入れると全店舗で品切れ表示になります
  5. 保存ボタンを押す

メニューの削除 ​

  1. 削除したいメニューの削除ボタンを押す
  2. 確認ダイアログが表示される：「このメニューを削除すると、全店舗のオーバーライド設定も削除されます」
  3. OKを押すと削除される

注意

メニューテンプレートを削除すると、そのメニューに対する全店舗のオーバーライド設定（価格変更、品切れ状態等）も一緒に削除されます。元に戻せません。

品切れのトグル ​

メニュー一覧の各メニューに品切れボタンがあります。

  • グレー（通常）→ クリックすると赤（品切れ）に変わる
  • 赤（品切れ）→ クリックするとグレー（通常）に戻る
  • 品切れにすると、セルフオーダー画面・ハンディPOS画面で即座に「品切れ」表示になり、お客さんは注文できなくなります
  • メニューバージョンが自動更新され、お客さんの画面は2秒以内に品切れ状態が反映されます

カテゴリでフィルター ​

メニュー一覧の上部にカテゴリフィルターのドロップダウンがあります。カテゴリを選択すると、そのカテゴリのメニューだけが表示されます。「全て」を選択すると全メニューが表示されます。

店舗メニューオーバーライド ​

店舗メニューオーバーライドでは、本部テンプレートのメニューを店舗ごとにカスタマイズします。価格の変更、メニューの非表示、店舗固有の画像設定が可能です。

サブタブ：「店舗メニュー」

店舗を選択してください

この画面は店舗単位で操作します。ヘッダーの店舗セレクターで、操作したい店舗を選択してから開いてください。

画面の見方 ​

テーブル形式で一覧表示されます。各行に以下の情報があります：

列説明
画像店舗固有の画像。設定がなければ本部テンプレートの画像
メニュー名名前（日本語/英語）
カテゴリ所属カテゴリ名
価格店舗価格（上書きされている場合はその価格、なければ本部価格）
非表示この店舗でメニューを非表示にするかどうか。非表示の行は半透明（薄い表示）になります
品切れこの店舗での品切れ状態
操作編集・削除ボタン

「本部品切れ」バッジが表示されている場合、本部テンプレート側で品切れに設定されているため、店舗側で解除できません。

メニューの価格を変更する ​

  1. 変更したいメニューの編集ボタンを押す
  2. モーダルが開く
  3. 価格欄に新しい価格を入力する（空欄にすると本部価格に戻ります）
  4. 保存ボタンを押す

メニューを非表示にする ​

この店舗では提供しないメニューを非表示にできます。

  1. メニュー一覧で、非表示にしたいメニューの非表示トグルを押す
  2. メニューが半透明表示になり、セルフオーダー画面に表示されなくなります
  3. もう一度押すと表示に戻ります

品切れにする ​

  1. メニュー一覧で、品切れにしたいメニューの品切れトグルを押す
  2. 赤いバッジが表示され、セルフオーダー画面で品切れ表示になります
  3. もう一度押すと品切れを解除できます

カロリー・アレルゲン情報を設定する ​

メニューの編集モーダルで、以下を設定できます：

項目説明入力方法
カロリーkcal単位の数値数値入力（例：450）
アレルゲン含まれるアレルゲンチェックボックスから選択

設定可能なアレルゲン（全25項目）： 卵、乳、小麦、えび、かに、そば、落花生、アーモンド、カシューナッツ、くるみ、ごま、さけ、さば、いか、あわび、いくら、大豆、まつたけ、もも、りんご、キウイ、バナナ、オレンジ、ゼラチン、鶏肉、牛肉、豚肉

設定したカロリーとアレルゲンはセルフオーダー画面のメニュー詳細に表示されます。

オプショングループを紐付ける ​

メニューの編集モーダルの下部に、オプショングループの紐付け設定があります。

  1. 編集モーダルを開く
  2. 下部の「オプション」セクションを確認する
  3. 紐付けたいオプショングループにチェックを入れる
  4. 「必須」チェックボックス：チェックを入れると、お客さんは注文時にこのオプションを必ず選択する必要があります
  5. 保存を押す

店舗固有の画像を設定する ​

  1. 編集モーダルを開く
  2. 画像アップロードエリアに画像をドラッグ＆ドロップ、またはクリックして選択
  3. プレビューが表示される
  4. 保存を押す
  5. この店舗では、本部テンプレートの画像の代わりにこの画像が表示されます

店舗限定メニュー ​

店舗限定メニューは、特定の店舗でのみ提供するオリジナルメニューです。本部テンプレートには存在しない、その店舗だけのメニューを作成できます。

サブタブ：「限定メニュー」

限定メニューの作成 ​

  1. 「メニュー追加」ボタンを押す
  2. モーダルが開く
  3. 以下を入力する：
項目必須説明
メニュー名✓日本語の表示名
メニュー名（英語）—英語の表示名
カテゴリ✓ドロップダウンから選択
価格（税込）✓税込価格（円）
説明文—メニューの詳細説明
画像—メニュー写真
カロリー—kcal（数値）
アレルゲン—25項目のチェックボックスから選択
オプショングループ—紐付けるオプション
  1. 保存ボタンを押す
  2. メニュー一覧に「限定」バッジ付きで追加される

限定メニューはセルフオーダー画面で通常のメニューと同様に表示されます（「限定」バッジは管理画面のみの表示で、お客さん側には表示されません）。

品切れにする ​

テンプレートメニューと同様に、品切れトグルボタンで切り替えます。

オプション・トッピング ​

オプショングループは、メニューに付加する選択肢（トッピング、サイズ、焼き加減など）を管理する機能です。

サブタブ：「オプション管理」

オプショングループの作成 ​

  1. 「グループ追加」ボタンを押す
  2. モーダルが開く
  3. 以下を入力する：
項目必須説明入力例
グループ名✓日本語名サイズ
グループ名（英語）—英語名Size
選択タイプ✓「単一選択」または「複数選択」単一選択
最小選択数—必須選択の最小数1
最大選択数—選択できる最大数1
  1. 保存ボタンを押す

選択タイプの違い：

タイプ動作自動設定例
単一選択ラジオボタン形式。1つだけ選択最小=1、最大=1サイズ（S/M/L）
複数選択チェックボックス形式。0〜N個選択最小=0、最大=3トッピング（チーズ/卵/ベーコン）

選択肢の追加 ​

オプショングループを作成したら、そのグループに選択肢を追加します。

  1. グループのカード内にある**「選択肢追加」ボタン**を押す
  2. モーダルが開く
  3. 以下を入力する：
項目必須説明入力例
選択肢名✓日本語名Lサイズ
選択肢名（英語）—英語名Large
価格差（円）—追加料金。マイナスも可+200
デフォルト—チェックすると初期選択される—
  1. 保存を押す

グループの有効/無効 ​

グループの編集モーダルに「有効」チェックボックスがあります。無効にすると、そのグループが紐付けられたメニューでもオプション選択が表示されなくなります。

使いどころ

季節限定のトッピングを一時的に無効にしたい場合や、在庫切れのオプションを停止したい場合に使います。紐付け自体を削除する必要はありません。

カロリー・アレルゲン情報 ​

カロリーとアレルゲンの設定方法は「店舗メニューオーバーライド」の編集モーダルで行います。詳細は4.3 店舗メニューオーバーライドを参照してください。

設定した情報は、セルフオーダー画面のメニュー詳細ポップアップに表示されます。アレルゲンはアイコン付きで表示され、お客さんがアレルギーを持つ食材を含むメニューを避けるのに役立ちます。

今日のおすすめ ​

「今日のおすすめ」は、日替わりで特定のメニューをセルフオーダー画面の上部に目立つように表示する機能です。

サブタブ：「店舗メニュー」内のおすすめセクション

おすすめの追加 ​

  1. 「店舗メニュー」サブタブを開く
  2. おすすめセクションのフォームが表示されている
  3. 以下を設定する：
項目必須説明
メニュー✓ドロップダウンからメニューを選択（テンプレート＋限定メニュー両方表示。限定メニューは [限定] タグ付き）
バッジ種別✓おすすめ / 人気 / 新着 / 限定 / 本日限り の5種類から選択
コメント—最大200文字のひとこと説明（例：「本日入荷の新鮮素材で作りました」）
  1. 追加ボタンを押す
  2. おすすめ一覧にカードが追加される

おすすめの削除 ​

  1. 削除したいおすすめカードの**×ボタン**を押す
  2. 確認ダイアログでOKを押す
  3. おすすめから削除される

自動リセット

おすすめは日付ごとに管理されています。翌日になると自動的にリセットされるわけではないので、毎日手動で更新する必要があります。前日のおすすめを削除し、新しいおすすめを追加してください。

品切れ管理 ​

品切れの設定は以下の3箇所から行えます：

場所範囲操作者
メニューテンプレート（カテゴリ管理タブ）全店舗で品切れマネージャー以上
店舗メニューオーバーライド（店舗メニュータブ）その店舗のみ品切れマネージャー以上
KDS画面その店舗のみ品切れKDS操作者

品切れ設定の反映は即時です。セルフオーダー画面は2秒ごとにメニューバージョンをチェックしており、品切れ設定が変更されるとお客さんの画面に自動反映されます。

在庫管理との連動

在庫管理機能（proプラン以上）を使用している場合、食材の在庫が0になるとそれを使用するメニューが自動的に品切れ表示になります。

メニュー多言語対応 ​

proプラン以上で利用できます。メニューを指定した言語に一括翻訳する機能です。

  1. 「店舗メニュー」サブタブを開く
  2. 「翻訳」ボタンを押す
  3. 翻訳先の言語を選択する
  4. 全カテゴリ・全メニュー名・全説明文・全オプション名が一括でAI翻訳されます
  5. 翻訳結果はキャッシュに保存され、セルフオーダー画面でお客さんの端末の言語設定に応じて自動表示されます

翻訳の仕組み

翻訳にはGemini 2.0 Flash APIを使用しています。30件ごとにバッチ処理されます。翻訳結果は menu_translations テーブルに保存（キャッシュ）され、同じメニューの再翻訳はキャッシュが使われます。メニュー名を変更した場合は再翻訳が必要です。

CSV一括インポート・エクスポート ​

メニュー関連のデータをCSVファイルで一括管理できます。

エクスポート（書き出し） ​

  1. 対象のサブタブを開く（店舗メニュー、限定メニュー等）
  2. 「CSVエクスポート」ボタンを押す
  3. CSVファイルがダウンロードされます
  4. ファイルはUTF-8（BOM付き）エンコーディングで、Excelでそのまま開けます

インポート（読み込み） ​

  1. 対象のサブタブを開く
  2. 「CSVインポート」ボタンを押す
  3. CSVファイルを選択する
  4. インポートが実行される
  5. 結果が表示される：作成された件数、更新された件数、エラーがあればエラー内容

CSVインポートの注意点

  • インポートはUPSERT方式です（既存のデータがあれば更新、なければ新規作成）
  • トランザクション内で実行されるため、途中でエラーがあった場合は全件ロールバックされます
  • エラーがあった場合、「行2: カテゴリが見つかりません」のように行番号とエラー内容が表示されます
