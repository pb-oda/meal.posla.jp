# U-2 プリンタ連携ガイド

## 対応機種 (2026-04-18 時点)

POSLA は以下のレシートプリンタに対応:

| 機種 | メーカー | 接続 | 価格目安 | ドライバ |
|---|---|---|---|---|
| **mC-Print3** | Star Micronics | Wi-Fi / Ethernet | ¥40,000前後 | **不要 (HTTP)** |
| **TM-m30III** | Epson | Wi-Fi / Ethernet | ¥45,000前後 | **不要 (HTTP)** |

どちらも **HTTP API** 標準搭載で、POSLA 側でドライバインストール不要。
店舗 LAN に Wi-Fi / Ethernet で接続するだけで動作します。

## 推奨構成

- **カウンター / レジ側**: 80mm 幅 (レシート)
- **キッチン側**: 58mm 幅 (注文伝票) — 任意

## 初期設定手順

### 1. プリンタを店舗 LAN に接続

#### Star mC-Print3
1. 電源ON、FEED ボタン長押しで設定シート印刷
2. 初期IP (DHCP) が印字される → 固定 IP に変更推奨
3. Web 管理画面: `http://{IP}` からアクセス

#### Epson TM-m30III
1. 電源ON、FEED ボタン長押しでIPアドレス印字
2. `http://{IP}` の EpsonNet Config から固定 IP 設定

### 2. POSLA 側設定

1. 管理画面 → 店舗設定 → プリンタ設定
2. **プリンタ種別**を選択 (Star / Epson / ブラウザ / 無効)
3. **プリンタ IP** を入力 (例: `192.168.1.50`)
4. **ポート**: 通常は `80` (HTTP 標準)
5. **用紙幅**: 80 or 58 (mm)
6. **自動印刷**:
   - キッチン伝票: 注文確定時に自動印刷 (ON/OFF)
   - レシート: 会計完了時に自動印刷 (ON/OFF)
7. 「テスト印刷」ボタンで動作確認

### 3. 動作確認

- キッチン伝票 (注文確定時に自動印刷)
- レシート (会計時に自動印刷)
- ラベル印刷 (テイクアウト注文受取時)

## トラブルシューティング

| 症状 | 原因 | 対処 |
|---|---|---|
| 印刷されない | プリンタとブラウザが別 VLAN | 同一 LAN に接続 |
| `network_error` | プリンタ電源OFF / IP違い | 接続と IP 再確認 |
| `http_404` | HTTP API 未起動 | プリンタ管理画面で WebPRNT/ePOS 有効化 |
| 文字化け | 文字コード不一致 | UTF-8 対応ファームウェアに更新 |
| 遅い | Wi-Fi 干渉 | Ethernet 接続に変更 |

## ブラウザ標準印刷 (デフォルト) の場合

プリンタ未設定店舗では「ブラウザ」モードで `window.print()` ダイアログが出ます。
初期運用はこれで十分。お客様が増えたタイミングで実機導入を検討してください。
