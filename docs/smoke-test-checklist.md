# POSLA リリース前 Smoke Test チェックリスト

最終更新: 2026-04-29 / 対象: POSLA通常レジ支払い詳細記録 + レジ締め照合 + 決済方針統一 + A5 LP 含む α-1 リリース版

## 実施方法

1. 本チェックリストは**手動実施**。コーダーが作成 → Hiro が手元で実行
2. 各項目を **✅/❌** でマーク。❌ が出たら該当コミット/ログを添えてレポート
3. 本番デプロイ前 or 大きな機能追加後に必ず実施
4. 完了まで目安: **60-90 分** (一人実施)

- **Critical**: マルチテナント境界 (11), レートリミット (12) は **100% 通過必須**

- **Medium**: それ以外は **5件以下の失敗まで許容** (要 Hiro 判断)
