# 15. オーナーダッシュボード

オーナーダッシュボードは、複数店舗の管理、本部機能、ユーザー管理、契約・決済設定を行う画面です。オーナーロールのユーザーが利用します。

## 画面の構成（10の主要機能）

最新のオーナーダッシュボードは、左側のナビゲーションから以下の10個の機能に素早くアクセスできます。

| タブ名称 | 内容 |
|---|---|
| **クロス店舗レポート** | 全店舗の売上、客数、客単価、回転率を横並びで比較分析します。 |
| **ABC分析** | 全店舗または指定店舗の商品別貢献度をA/B/Cランクで可視化します。 |
| **シフト概況** | 各店舗の人件費、労働時間、充足率、ヘルプ要請状況を把握します。 |
| **本部メニュー** | チェーン共通のマスタメニューを作成し、各店舗へ一括配信します。 |
| **店舗管理** | 新規店舗の追加、既存店舗の設定変更、休業・削除を管理します。 |
| **ユーザー管理** | オーナー、マネージャー、スタッフ、端末用アカウントを作成・編集します。 |
| **AIアシスタント** | SNS投稿案の作成、売上分析、需要予測、競合調査などをAIが支援します。 |
| **契約・プラン** | POSLAの月額利用料、トライアル期間、アドオン（本部配信）を管理します。 |
| **決済設定** | Stripe Connect の連携や、テナント独自の決済キーを設定します。 |
| **外部連携** | LINE公式アカウントとの接続設定、友だち紐付け、スマレジ連携を管理します。 |

## 外部連携（LINE / スマレジ）

「外部連携」タブでは、店舗運営を加速させる外部サービスとの接続を一元管理します。

- **LINE公式アカウント**: 予約確定、リマインド、テイクアウト準備完了などの通知（LINE v1）を自動送信するための設定を行います。
    - ※リッチメニュー、LIFF、メッセージ配信、チャットボット等の機能（LINE v2）は、現在開発中のバックログであり未提供です。
- **スマレジ連携**: POSLAでの注文データをスマレジへ自動送信し、売上管理を統合します。

## 契約と決済

「契約・プラン」および「決済設定」はオーナーのみがアクセス可能な重要項目です。
- クレジットカードの更新やプランの変更は「契約・プラン」から行います。
- お客様のセルフ決済を有効にするための Stripe 連携は「決済設定」から行います。

## 店舗を追加する

1. 店舗管理を開きます。
2. 店舗名、住所、営業時間、税率などを入力します。
3. 保存後、必要なマネージャーを所属させます。
4. 店舗側でメニュー、テーブル、決済、プリンタを設定します。

## ユーザー管理

オーナーは全店舗のユーザーを管理できます。退職者や異動者がいる場合は、速やかにアカウントを無効化または所属店舗を変更してください。

## 複数店舗レポート

店舗別売上、商品傾向、スタッフ実績を横断して確認できます。期間と対象店舗をそろえて比較してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 店舗が表示されない | 店舗の登録状態と契約内容を確認してください。 |
| ユーザーがログインできない | 所属店舗、ロール、パスワードを確認してください。 |
| 本部メニューが使えない | 契約プランまたはオプションを確認してください。 |
| 決済設定が進まない | 入力内容を確認し、必要に応じてPOSLAサポートへお問い合わせください。 |
