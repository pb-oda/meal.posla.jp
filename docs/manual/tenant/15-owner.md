# 15. オーナーダッシュボード

オーナーダッシュボードは、複数店舗の管理、本部機能、ユーザー管理、契約・決済設定を行う画面です。オーナーロールのユーザーが利用します。

## 主な機能

| 機能 | 内容 |
|---|---|
| 店舗管理 | 店舗の追加、編集、状態確認 |
| ユーザー管理 | マネージャー、スタッフ、端末専用アカウントの管理 |
| クロス店舗レポート | 複数店舗の売上や実績を横断比較 |
| ABC分析 | 売上貢献度をもとに商品の傾向を確認 |
| 本部メニュー | 多店舗向けのメニュー一括管理 |
| 決済設定 | Stripeや決済関連情報の確認・設定 |

## 店舗を追加する

1. 店舗管理を開きます。
2. 店舗名、住所、営業時間、税率などを入力します。
3. 保存後、必要なマネージャーを所属させます。
4. 店舗側でメニュー、テーブル、決済、プリンタを設定します。

## ユーザー管理

オーナーは全店舗のユーザーを管理できます。退職者や異動者がいる場合は、速やかにアカウントを無効化または所属店舗を変更してください。

## 複数店舗レポート

店舗別売上、商品傾向、スタッフ実績を横断して確認できます。期間と対象店舗をそろえて比較してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 店舗が表示されない | 店舗の登録状態と契約内容を確認してください。 |
| ユーザーがログインできない | 所属店舗、ロール、パスワードを確認してください。 |
| 本部メニューが使えない | 契約プランまたはオプションを確認してください。 |
| 決済設定が進まない | 入力内容を確認し、必要に応じてPOSLAサポートへお問い合わせください。 |
