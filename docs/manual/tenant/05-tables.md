# 5. テーブル・フロアマップ

テーブル管理では、店舗の客席、QRコード、フロアマップ、着席状態を管理します。セルフオーダーを始める前に必ず設定してください。

## テーブル登録

1. 管理画面でテーブル管理を開きます。
2. テーブル名、座席数、表示順を入力します。
3. 保存後、必要に応じてQRコードを発行します。
4. テーブル番号と実際の席が一致しているか確認します。

## QRコードの掲示

発行したQRコードを印刷し、該当テーブルに掲示します。お客様が読み取ると、そのテーブルのセルフオーダー画面が開きます。

## フロアマップ

フロアマップでは、実際の席配置に近い形でテーブルを配置できます。スタッフが空席、着席、会計待ちなどの状況を把握しやすくなります。

## テーブルの開放と終了

営業中は、お客様が着席したタイミングでテーブルを開放します。退店後は会計完了と片付けを確認し、次のお客様に使える状態にしてください。

## 相席・分割利用

同じテーブルを複数グループで使う運用や、注文を分けたい場合は、店舗の運用ルールに合わせて管理してください。迷う場合はPOSLAサポートへご相談ください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| QRコードを読み取れない | 印刷のかすれ、掲示位置、カメラの距離を確認してください。 |
| 違うテーブルが開く | QRコードを再発行し、貼り間違いがないか確認してください。 |
| フロアマップが実席と合わない | テーブル位置をドラッグして調整してください。 |
| 注文が別テーブルに入る | テーブル番号とQRコードの対応を確認してください。 |
