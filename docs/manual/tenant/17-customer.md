# 17. セルフオーダー画面

セルフオーダー画面は、お客様がテーブルQRコードから開く注文画面です。店舗スタッフは、表示内容と注文の流れを理解しておくと問い合わせ対応がしやすくなります。

## お客様の注文手順

1. テーブルのQRコードをスマートフォンで読み取ります。
2. メニューを選びます。
3. 数量やオプションを選択します。
4. カート内容を確認します。
5. 注文を送信します。
6. 必要に応じてスタッフ呼び出しやAIウェイターを利用します。

## 店舗側で確認すること

- QRコードが正しいテーブルに貼られていること
- メニュー、価格、写真、売り切れ状態が正しいこと
- 注文がKDSとレジに反映されること
- 多言語表示の内容が大きくずれていないこと

## AIウェイター

お客様はメニューについて質問できます。アレルギー、提供可否、価格など重要な内容は、店舗スタッフが最終確認してください。

## スタッフ呼び出し

お客様から呼び出しが入った場合は、店舗のルールに従って対応してください。呼び出し内容が分かる場合は、先に内容を確認してから向かうとスムーズです。

## 困ったとき

| 症状 | 対処 |
|---|---|
| メニューが表示されない | QRコード、店舗設定、メニュー公開状態を確認してください。 |
| 注文できない | Wi-Fi、売り切れ状態、テーブル開放状態を確認してください。 |
| 表示言語が違う | 画面上の言語切替を確認してください。 |
| 金額が違う | 商品価格、オプション、税区分を確認してください。 |
