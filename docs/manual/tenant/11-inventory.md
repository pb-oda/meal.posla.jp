# 11. 在庫・レシピ管理

在庫・レシピ管理では、原材料、在庫数、使用量、棚卸し、AI需要予測を管理します。品切れ防止と仕入れ判断に役立ちます。

## 原材料を登録する

1. 原材料名、単位、現在庫、低在庫の目安を登録します。
2. 仕入れ単価や保管場所を必要に応じて入力します。
3. 棚卸し時に実在庫を更新します。

## レシピを設定する

商品ごとに使用する原材料と数量を登録します。注文数に応じて在庫を確認しやすくなります。

## 棚卸し

営業終了後や月末など、店舗の運用に合わせて棚卸しを行います。実在庫とシステム上の在庫がずれている場合は、理由を確認してください。

## AI需要予測

過去の販売傾向をもとに、仕込みや発注の参考値を確認できます。天候、イベント、臨時休業など、システムに反映されない事情は店舗判断で調整してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 在庫数が合わない | 棚卸し、廃棄、仕入れ、手入力の履歴を確認してください。 |
| 品切れ表示が合わない | 商品とレシピの紐づけ、低在庫の目安を確認してください。 |
| 需要予測が極端 | 休業日、イベント、異常値が含まれていないか確認してください。 |
| CSVで文字化けする | UTF-8形式で保存してから取り込んでください。 |
