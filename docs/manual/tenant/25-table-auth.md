# 25. テーブル開放認証

テーブル開放認証は、お客様がQRコードから注文する前に、対象テーブルが利用可能な状態か確認する仕組みです。誤注文やいたずら注文を防ぐために利用します。

## 基本の考え方

- お客様が着席したら、店舗側でテーブルを利用可能にします。
- お客様は該当テーブルのQRコードから注文します。
- 退店後は会計完了と片付けを確認し、テーブルを終了します。

## 営業中の流れ

1. お客様を席へ案内します。
2. 必要に応じてテーブルを開放します。
3. お客様がQRコードから注文します。
4. 会計完了後、テーブルを終了します。
5. 次のお客様用に状態を整えます。

## 注意点

- QRコードの貼り間違いに注意してください。
- テーブル終了を忘れると、次のお客様の注文と混ざる可能性があります。
- 予約来店の場合は、予約情報とテーブル状態を確認してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| お客様が注文画面に進めない | テーブルが開放されているか確認してください。 |
| 別テーブルの画面が開く | QRコードの貼り間違いを確認してください。 |
| 前のお客様の情報が残る | 会計完了とテーブル終了を確認してください。 |
| 予約と自動で合わない | 予約日時、人数、テーブル状態を確認してください。 |
