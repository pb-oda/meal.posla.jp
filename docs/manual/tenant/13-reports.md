# 13. レポート

レポートでは、売上、会計、商品、客席、スタッフ、在庫、予約などの状況を確認できます。日々の振り返りと経営判断に活用してください。

## 主なレポート

| レポート | 確認できること |
|---|---|
| **週次サマリー** | 売上レポートの最上部にある、直近1週間の売上・客数・客単価・人気商品の概要。 |
| 売上レポート | 日別、期間別、支払い方法別の売上 |
| レジレポート | 現金、入金、出金、差額 |
| 商品レポート | 売れ筋、ABC分析、併売傾向 |
| 客席レポート | 回転率、滞在時間、テーブル別傾向 |
| スタッフレポート | 勤怠、操作履歴、担当別の実績 |
| 満足度分析 | 評価件数、平均評価、低評価の理由・品目・時間帯 |
| 予約レポート | 予約件数、来店数、キャンセル傾向 |

## 満足度分析

管理画面 → レポート → 満足度分析 から開きます。お客様のセルフ注文画面で送信された評価（1〜5）を集計し、店舗改善に活用できます。

- **サマリー**: 評価件数、平均評価、低評価件数（1〜2）、低評価率、高評価件数（4〜5）
- **低評価の理由ランキング**: 提供が遅かった／味／温度／量／注文内容の相違／接客／清潔感／価格／その他
- **低評価が出た品目ランキング**: どの品目に低評価が集中しているか
- **時間帯別 低評価**: ピーク時に提供品質が落ちていないか確認
- **直近の低評価一覧**: 日時・品目・評価・理由・自由コメント・テーブル・注文ID

::: tip 低評価は改善のヒントとして活用してください
低評価はスタッフ個人の責任ではなく、「品目・時間帯・店舗運営の改善点」として活用する前提で設計しています。顧客セルフ注文ではスタッフ担当者が特定されないケースも多いため、まずは品目・時間帯・オペレーションの見直しをお勧めします。
:::

オーナー画面の「全店舗横断レポート」にも、全店舗合算の満足度サマリー・店舗別平均評価・低評価が多い品目・理由ランキングが表示されます。

## 期間の選び方

日報は営業終了後、週報は週明け、月報は月末または翌月初に確認する運用を推奨します。深夜営業の店舗では、営業日区切り時刻に注意してください。

## CSV出力

必要に応じてCSVを出力し、会計ソフトや表計算ソフトで集計できます。出力後は個人情報や売上情報の取り扱いに注意してください。

## 活用例

- 売れ筋商品を確認して仕入れ量を調整する
- 回転率の低い時間帯を見つけて人員配置を見直す
- 返金やレジ差額を確認して運用改善につなげる
- スタッフ別の実績を評価や教育に活用する

## 困ったとき

| 症状 | 対処 |
|---|---|
| 数値が合わない | 期間、営業日区切り、会計済み状態を確認してください。 |
| CSVが開けない | UTF-8対応のソフトで開いてください。 |
| 特定店舗が出ない | 店舗選択と閲覧権限を確認してください。 |
| レジ差額がある | 入金、出金、返金、現金残高を確認してください。 |
