# 13. レポート

管理ダッシュボードの「レポート」タブでは、売上・レジ・回転率・スタッフ評価・併売分析のレポートを閲覧できます。マネージャー以上のロールで利用できます。

基本レポート（売上レポート）はstandardプランで利用でき、高度レポート（回転率・スタッフ評価・併売分析）はproプラン以上で利用できます。

## 13.1 共通操作

### 期間の選択

全てのレポートで期間を指定できます。プリセットボタンとカスタム範囲が用意されています。

| プリセット | 期間 |
|-----------|------|
| 今日 | 本日の営業日 |
| 昨日 | 前日の営業日 |
| 今週 | 今週（月曜〜日曜） |
| 今月 | 今月（1日〜末日） |

カスタム範囲では開始日と終了日を自由に指定できます。

### CSVエクスポート

多くのレポートでCSVエクスポートに対応しています。データを外部ツール（Excel等）で分析する際に利用します。

## 13.2 売上レポート

「レポート」タブ → 「売上レポート」サブタブ

### サマリーカード

| 指標 | 説明 |
|------|------|
| 注文件数 | 期間内の総注文数 |
| 売上合計 | 期間内の総売上金額 |
| 平均注文額 | 売上合計 ÷ 注文件数 |

### サービス時間分析

注文から提供までの時間を分析します。

| 指標 | 説明 |
|------|------|
| 平均 | 全注文の平均サービス時間（秒） |
| 中央値 | 中央値 |
| 最短 | 最も早かったサービス時間 |
| 最長 | 最も遅かったサービス時間 |

サービス時間は、注文の`created_at`から`served_at`（または`ready_at`）までの差分で計算されます。

### 品目ランキング

売上金額順のトップ20品目が表示されます。

| 列 | 説明 |
|----|------|
| 品名 | メニュー名 |
| 数量 | 販売数 |
| 売上 | その品目の総売上 |

### 時間帯別売上

1時間ごとの注文数と売上をバーチャートで表示します。

| 列 | 説明 |
|----|------|
| 時間帯 | 0時〜23時 |
| 注文数 | その時間帯の注文件数 |
| 売上 | その時間帯の売上金額 |

### テーブル別売上

テーブルごとの注文数と売上を表示します。

## 13.3 レジレポート

「レポート」タブ → 「レジ分析」サブタブ

### サマリーカード

| 指標 | 説明 |
|------|------|
| レジ開け金額 | 営業開始時のつり銭準備金 |
| 現金売上 | 現金決済の合計 |
| 入金 | 営業中の入金合計 |
| 出金 | 営業中の出金合計 |
| 予想残高 | 開始金額 + 現金売上 + 入金 − 出金 |
| レジ閉め金額 | 営業終了時の実際の残高 |
| 過不足 | レジ閉め金額 − 予想残高 |

**過不足の色分け：**
- マイナス（不足）：赤色
- プラス（過剰）：青色

### 支払方法別内訳

| 列 | 説明 |
|----|------|
| 支払方法 | 現金 / クレジットカード / QR |
| 件数 | 決済件数 |
| 合計 | 決済金額合計 |

### キャッシュログ詳細

全てのレジ操作の時系列記録です。

| 列 | 説明 |
|----|------|
| 時刻 | 操作日時 |
| 種別 | レジ開け / レジ締め / 入金 / 出金 / 現金売上 |
| 操作者 | スタッフ名 |
| 金額 | — |
| メモ | — |

## 13.6 併売分析（proプラン以上）

「レポート」タブ → 「併売分析」サブタブ

### アソシエーションルール

一緒に注文される品目の組み合わせを分析します（マーケットバスケット分析）。

**最小サポート閾値の設定：** 2 / 3 / 5 / 10 から選択（最低何回以上の同時注文があれば表示するか）

| 列 | 説明 |
|----|------|
| 品目A | — |
| 品目B | — |
| サポート | 全注文中での同時出現回数 |
| 信頼度（A→B） | Aを注文した人がBも注文する確率 |
| 信頼度（B→A） | Bを注文した人がAも注文する確率 |
| リフト値 | 相関の強さ |

**リフト値の解釈：**
- **1.5以上（緑色）：** 強い正の相関（一緒に注文されやすい）
- **1.0前後：** 無相関
- **1.0未満（赤色）：** 負の相関（一緒に注文されにくい）

### 品目頻度ランキング

注文頻度の上位20品目をバーチャートで表示します。

::: tip 併売分析の活用
- リフト値の高い組み合わせをセットメニューとして提案
- 「一緒にいかがですか？」のレコメンド根拠に活用
- メニュー配置（カテゴリ順やおすすめ表示）の最適化に活用
:::
