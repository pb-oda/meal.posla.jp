# 2. ログインとアカウント

この章では、POSLAへのログイン、パスワード変更、スタッフアカウント、端末専用アカウントの扱いを説明します。

## ログイン方法

1. POSLAのログイン画面を開きます。
2. ユーザー名とパスワードを入力します。
3. 「ログイン」を押します。
4. ロールに応じた管理画面が表示されます。

メールアドレスは必須ではありません。店舗内で管理しやすいユーザー名を使ってください。

## パスワードの管理

- パスワードは8文字以上で、英字と数字を含めてください。
- 初期パスワードは、初回ログイン後に本人が変更する運用を推奨します。
- パスワードを忘れたスタッフは、オーナーまたはマネージャーに再設定を依頼してください。
- オーナーがログインできない場合は、POSLAサポートへお問い合わせください。

## 1人1アカウントのルール

POSLAでは、スタッフごとに個別アカウントを作成してください。共有アカウントは使用しないでください。

| 理由 | 説明 |
|---|---|
| 勤怠管理 | 出勤・退勤を個人ごとに記録するため |
| 会計操作 | 返金や現金操作の担当者を明確にするため |
| 監査ログ | いつ誰が操作したかを確認できるようにするため |
| シフト管理 | 希望提出や実績集計を個人単位で行うため |

## アカウント作成権限

| 操作者 | 作成できるアカウント | 対象範囲 |
|---|---|---|
| オーナー | マネージャー、スタッフ、端末専用アカウント | 全店舗 |
| マネージャー | スタッフ、端末専用アカウント | 自分の所属店舗 |
| スタッフ | 作成不可 | なし |

## 端末専用アカウント

KDS、POSレジ、ハンディなどの店舗備品には、端末専用アカウントを使用します。端末専用アカウントは人ではなく機器を表すため、シフト、勤怠、人件費、スタッフ評価の対象にはなりません。

## PINについて

POSレジでの会計、返金、レジ開け、レジ閉め、現金入金、現金出金では、担当スタッフのPIN入力が必要です。PINにより、端末専用アカウントでログインしている場合でも、実際に操作したスタッフを記録できます。

## セッションとログアウト

管理画面を使う共有PCでは、作業終了後にログアウトしてください。スタッフ個人アカウントを共有PCに残したままにしないためです。

KDS、POSレジ、ハンディの店舗備品端末は、端末専用アカウントで常時ログインしておく運用です。日次閉店時はログアウトではなく、画面ロック、充電、通知音、翌日使う端末モードを確認してください。端末を紛失した、端末を入れ替える、店舗から外す場合は、対象の端末専用アカウントを無効化またはパスワード変更してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| ログインできない | ユーザー名、パスワード、半角全角、Caps Lockを確認してください。 |
| パスワードを忘れた | 上司に再設定を依頼してください。 |
| 画面が真っ白になる | ブラウザを再読み込みしてください。改善しない場合は別端末でお試しください。 |
| 権限が不足している | オーナーまたはマネージャーに権限を確認してください。 |
