# 20. Googleレビュー連携

Googleレビュー連携では、満足度の高いお客様をGoogleレビューへ案内し、店舗の評価向上につなげます。

## 基本の流れ

1. 店舗設定でGoogleレビュー連携を確認します。
2. レビュー案内の表示内容を設定します。
3. 会計後やセルフオーダー画面で満足度評価を受け付けます。
4. 高評価のお客様をGoogleレビューへ案内します。
5. 低評価のお客様には店舗内で改善につなげます。

## 運用時の注意

- レビュー投稿を強制しないでください。
- 見返りを条件にレビューを依頼しないでください。
- 低評価を隠す目的で不適切な誘導を行わないでください。
- お客様の声は、接客やメニュー改善の参考にしてください。

## 満足度評価

満足度評価は、店舗内での改善にも役立ちます。低評価が続く場合は、時間帯、商品、スタッフ体制などを確認してください。

低評価（1〜2）を選んだお客様には、差し支えない範囲で気になった点（提供時間／味／温度／量／注文内容／接客／清潔感／価格 など）をお伺いする画面が表示されます。任意入力で、お客様に負担をかけない設計です。

集計結果は管理画面 → レポート → 満足度分析 で確認できます（詳しくは「13. レポート」をご覧ください）。集計は店舗改善のために活用してください。低評価を「外部レビューに出ないようにする」ためではなく、品目・時間帯・運営の見直しに使う仕組みです。

## 困ったとき

| 症状 | 対処 |
|---|---|
| レビュー案内が出ない | 連携設定と表示タイミングを確認してください。 |
| リンク先が違う | GoogleビジネスプロフィールのURLを確認してください。 |
| 評価が偏る | 案内文や表示タイミングを見直してください。 |
