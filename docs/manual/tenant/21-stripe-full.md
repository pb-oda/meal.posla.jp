# 21. 決済設定

決済設定では、POSLAの月額契約、お客様からのカード決済、カードリーダー、返金に関する設定を確認します。決済機能は店舗の売上に直接関わるため、営業開始前に必ずテストしてください。

## 決済の種類

| 種類 | 内容 |
|---|---|
| POSLA利用料 | POSLAの月額利用料のお支払い |
| 店舗のカード決済 | お客様が店内またはオンラインで支払うカード決済 |
| カードリーダー決済 | POSレジと連携した対面カード決済 |
| 返金 | 会計後の返金処理 |

## 店舗側で確認すること

1. 契約プランと請求先を確認します。
2. 店舗で利用する決済方法を確認します。
3. カードリーダーを利用する場合は、接続とテスト決済を行います。
4. 返金時の承認ルールを店舗内で決めます。
5. レジ締め時に決済金額とPOSLA上の売上を確認します。

## カードリーダー利用時の注意

- レジ端末とカードリーダーを同じネットワークで利用してください。
- 営業前に充電、接続、テスト決済を確認してください。
- 決済完了前に画面を閉じないでください。
- エラーが出た場合は、二重決済になっていないか確認してください。

## 返金

返金は店舗責任者の確認後に行ってください。返金理由、対象注文、返金額、担当者を記録しておくと、後日の確認がスムーズです。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 決済が完了しない | カードリーダー、通信、カード状態を確認してください。 |
| 二重決済が心配 | 決済履歴を確認し、不明な場合はPOSLAサポートへお問い合わせください。 |
| 返金できない | 権限、対象注文、決済状態を確認してください。 |
| 契約内容を確認したい | オーナー画面またはPOSLAサポートで確認してください。 |
