# 26. プリンタ連携

プリンタ連携では、レシート、領収書、キッチン伝票などの印刷を行います。営業開始前に必ずテスト印刷を実施してください。

## 対応する印刷物

| 印刷物 | 主な利用場面 |
|---|---|
| レシート | 会計完了時 |
| 領収書 | お客様から依頼されたとき |
| キッチン伝票 | 厨房で紙伝票を併用する場合 |
| QRコード | テーブル、テイクアウト、予約案内 |

## 設定前に確認すること

- プリンタの電源が入っていること
- 用紙が入っていること
- 店舗ネットワークにつながっていること
- レジ端末からプリンタに接続できること
- レシートに表示する店舗名、住所、登録番号が正しいこと

## テスト印刷

設定後は、レシート、領収書、キッチン伝票をそれぞれテスト印刷してください。文字化け、余白、用紙幅、ロゴ、金額表示を確認します。

## 営業中の注意

- 用紙切れに備えて予備ロールを用意してください。
- プリンタの電源やネットワークが切れないようにしてください。
- 印刷できない場合の手書き対応を店舗内で決めておいてください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 印刷されない | 電源、用紙、ネットワーク、プリンタ選択を確認してください。 |
| 文字化けする | プリンタ設定と用紙幅を確認してください。 |
| 領収書の内容が違う | 店舗情報と領収書設定を確認してください。 |
| キッチン伝票が出ない | 対象商品と印刷先を確認してください。 |
