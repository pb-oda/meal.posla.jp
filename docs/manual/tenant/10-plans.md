# 10. プラン・コース管理

プラン・コース管理では、食べ放題、飲み放題、セットコース、予約コースなどを設定します。

## 時間制プラン

食べ放題や飲み放題など、利用時間とラストオーダーが決まっているメニューに利用します。

1. プラン名を入力します。
2. 利用時間、ラストオーダー、価格を設定します。
3. 対象商品または対象カテゴリを設定します。
4. テスト注文で時間表示と会計金額を確認します。

設定する主な項目:

| 項目 | 内容 |
|---|---|
| プラン名 | お客様画面やハンディに表示される名称 |
| 利用時間 | 何分利用できるか |
| ラストオーダー | 終了何分前に注文受付を止めるか |
| 価格 | プラン料金 |
| 対象メニュー | プランで注文できる品目 |
| 表示状態 | 販売中、非表示、停止など |

時間制プランを使う場合は、対象外の商品が通常注文として混ざらないか、会計時にプラン料金と単品料金が意図通りになるかを必ず確認してください。

## プラン専用メニューCSV

プランの対象メニューが多い場合は、CSVで一括更新できます。管理画面の **プラン・コース → 食べ放題プラン** から、プランCSVとメニューCSVを扱います。

プラン専用メニューCSVでは、プラン名、カテゴリ、品目名、説明、画像、表示順、有効/無効などをまとめて編集できます。インポート後は、対象プランで実際に注文できる品目と、対象外品目が注文できないことをテストしてください。

## コース料理

コース料理では、提供順や構成品目を管理します。予約管理と組み合わせる場合は、予約時に選べるコース内容も確認してください。

コース料理は、予約管理とKDSの両方に関係します。

| 項目 | 内容 |
|---|---|
| コース名 | 予約時や管理画面に表示 |
| 所要時間 | 予約枠や席利用時間の目安 |
| 構成 | 前菜、主菜、デザートなどの流れ |
| 対象メニュー | 実際に提供する品目 |
| 予約表示 | お客様予約画面で選べるか |

コース予約が入る店舗では、予約台帳、ハンディ、KDS、POSレジにコース名や予約文脈が表示されます。アレルギーや注意点がある場合は、コース内容と合わせて確認してください。

## 運用時の注意

- プラン料金と単品料金の二重計上がないか確認してください。
- ラストオーダー時刻はスタッフ全員に共有してください。
- 対象外の商品が注文できないかテストしてください。
- キャンペーン価格を設定する場合は、終了後に戻す運用を決めてください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| プランが表示されない | 販売状態、対象店舗、表示期間を確認してください。 |
| 金額が違う | プラン価格、オプション、割引を確認してください。 |
| ラストオーダー後に注文できる | プラン時間と端末時刻を確認してください。 |
| コース内容が違う | コース構成とメニュー登録を確認してください。 |
