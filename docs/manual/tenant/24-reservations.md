# 24. 予約管理

予約管理では、予約受付、予約台帳、空席確認、来店処理、キャンセル、予約金、コース予約を管理します。

## 予約受付の流れ

1. お客様が予約ページを開きます。
2. 日時、人数、連絡先、希望内容を入力します。
3. 店舗側の予約台帳に反映されます。
4. 来店時に予約を確認し、席へ案内します。
5. 必要に応じてテーブル開放や注文へ進みます。

## 予約台帳

予約台帳では、日付、時間、人数、氏名、連絡先、コース、ステータスを確認します。営業前に当日の予約を確認し、席と人員を準備してください。

## 予約ステータス

| ステータス | 意味 |
|---|---|
| 受付済み | 予約が登録された状態 |
| 来店済み | お客様が来店し、案内済みの状態 |
| キャンセル | 予約が取り消された状態 |
| 無断キャンセル | 予約時間を過ぎても来店がない状態 |

## 予約金

予約金を利用する場合は、対象コース、返金条件、キャンセルポリシーを事前に明確にしてください。お客様への案内文も確認してください。

## AI予約

AI予約は自然文から予約内容を読み取る補助機能です。最終的な日時、人数、コース、連絡先は必ず店舗側で確認してください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 予約が表示されない | 日付、店舗、ステータスを確認してください。 |
| 席が足りない | フロアマップ、予約人数、滞在時間を確認してください。 |
| キャンセル処理したい | 店舗のキャンセルポリシーに従って処理してください。 |
| 予約金の扱いが分からない | 決済履歴と店舗ルールを確認し、必要に応じてPOSLAサポートへお問い合わせください。 |
