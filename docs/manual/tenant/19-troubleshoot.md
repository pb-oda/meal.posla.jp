# 19. トラブルシューティング

問題が起きたときは、画面名、操作内容、表示されたメッセージ、発生時刻を確認してください。POSLAサポートへ連絡する際にも、この情報があると確認が早くなります。

::: tip まず最初に: 管理画面右下の「📚 サポートガイド」を開く
管理画面右下のオレンジ色の **「📚 サポートガイド」** ボタンから、よくある質問や対処方法をすぐに確認できます。以下の 5 入口で答えを探せます:

- **ホーム**: カテゴリ別ショートカット + 人気の質問 + エラー番号入力欄
- **よくある質問**: ログイン/注文/会計/印刷/予約/KDS/メニュー/シフト/テイクアウト/設定 のカテゴリ別 Q&A
- **キーワード検索**: 「会計」「パスワード」などで横断検索
- **症状からたどる**: 「KDSに出ない」等の症状から手順を表示
- **エラー番号**: `E3024` 等を入力すると意味と対処を即表示

まずはサポートガイドで該当する項目を探し、それでも解決しない場合に POSLAサポートへご連絡ください。
:::

## まず確認すること

1. インターネット接続を確認します。
2. ブラウザを再読み込みします。
3. 別の端末またはブラウザで試します。
4. ログイン中のユーザーと権限を確認します。
5. 店舗選択が正しいか確認します。

## よくある症状

| 症状 | 確認すること |
|---|---|
| ログインできない | ユーザー名、パスワード、アカウント状態 |
| 注文が届かない | Wi-Fi、KDS画面、店舗・テーブル設定 |
| 会計できない | 注文状態、支払い方法、PIN、決済端末 |
| メニューが出ない | 商品の販売状態、カテゴリ、店舗表示設定 |
| QRコードが読めない | 印刷状態、貼り間違い、テーブル開放状態 |
| レポートが合わない | 期間、営業日区切り、会計済み状態 |
| 画面に黄色いバナーで「古いデータを表示中」と出る | 通信が一時的に切れています。**操作は通信復帰後に行ってください**。詳細は『KDS デバイスガイド』「通信が一時的に切れた場合の挙動」 |

## POSLAサポートへ連絡する前に

以下を控えてください。

- 店舗名
- 操作した画面
- 操作したユーザー名
- 発生日時
- 表示されたメッセージ
- 可能であればスクリーンショット

## 緊急時の考え方

営業中に注文や会計が止まる場合は、まず店舗の代替運用に切り替えてください。紙伝票、手書き会計、手動レジ締めなどの緊急対応を行い、営業後にPOSLAサポートへ状況をご共有ください。

## 困ったとき

管理画面右下の **「📚 サポートガイド」** (オレンジ色の丸ボタン) も利用できます。FAQ / キーワード検索 / 症状フロー / エラー番号から、必要な案内をすぐに確認できます。
