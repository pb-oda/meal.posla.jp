# 9. テイクアウト管理

テイクアウト機能は**proプラン以上**で利用できます。お客さんがオンラインでテイクアウト注文を行い、店舗側で受付・調理・受取の管理を行います。

## 9.1 テイクアウト設定

テイクアウト機能を利用するには、まず店舗設定で有効化する必要があります。

管理ダッシュボードの「店舗設定」→「営業設定」から以下を設定します。

| 設定項目 | 初期値 | 説明 |
|---------|--------|------|
| テイクアウト有効 | OFF | ONにするとテイクアウト注文を受け付けます |
| 最低準備時間 | 30分 | 注文から受取までの最低時間。これより早い時間帯は選択できません |
| スロット容量 | 5件 | 15分間隔の各時間帯で受け付ける最大注文数 |
| 受付開始時間 | 10:00 | テイクアウト注文を受け付ける開始時刻 |
| 受付終了時間 | 20:00 | テイクアウト注文を受け付ける終了時刻 |
| オンライン決済 | OFF | ONにするとオンライン事前決済が可能になります（決済ゲートウェイの設定が必要） |

### テイクアウト用QRコード

テイクアウト設定画面には、お客さん向けのテイクアウト注文ページのQRコードが自動生成されます。

**QRコードのURL形式：**
```
https://あなたのドメイン/customer/takeout.html?store_id=店舗ID
```

**QRコードの操作ボタン：**
- URLをコピー
- QR画像をコピー（クリップボードAPI）
- QRコードを印刷（ブラウザ印刷ダイアログ）

::: tip QRコードの活用
テイクアウト用QRコードを店頭のポスターやSNS、チラシに掲載することで、お客さんが簡単にテイクアウト注文画面にアクセスできます。
:::

## 9.2 テイクアウト注文管理画面

管理ダッシュボードの「テイクアウト」タブを選択すると、テイクアウト注文の管理画面が表示されます。

### フィルター

| フィルター | 説明 |
|-----------|------|
| 日付 | 注文日を選択（初期値：今日） |
| ステータス | 全件 / 受付済み / 決済待ち / 調理中 / 準備完了 / 受取完了 |

### 注文一覧

各注文は以下の情報を持つカードとして表示されます。

| 表示要素 | 説明 |
|---------|------|
| 受取時間 | HH:MM形式（例：「14:30」） |
| ステータスバッジ | アニメーション付きで色分け |
| 注文ID | 先頭8文字（大文字）で表示 |
| お客さん名 | — |
| 電話番号 | — |
| 品目サマリー | 品名と数量のカンマ区切り |
| 合計金額 | — |
| メモ | 灰色背景で表示（ある場合のみ） |
| アクションボタン | ステータスに応じたボタン |

### ステータスの遷移と操作ボタン

| 現在のステータス | ボタン | 遷移先 |
|---------------|--------|--------|
| pending（受付済み） | 「調理開始」 | preparing |
| preparing（調理中） | 「準備完了」（緑色） | ready |
| ready（準備完了） | 「受取完了」（オレンジ色） | served |

**注意：** 「受取完了」に遷移すると、「店頭支払いの場合はレジで会計してください」というメッセージが表示されます。オンライン決済済みでない場合は、POSレジで会計処理を行ってください。

### 自動更新

テイクアウト注文一覧は**30秒ごと**に自動更新されます。

## 9.3 お客さん側のテイクアウト注文フロー

お客さんがテイクアウト用QRコードを読み取ると、以下の流れで注文できます。

1. **テイクアウト設定の読み込み：** 店舗のブランドカラー、ロゴ、営業時間が表示されます
2. **受取時間の選択：** 15分間隔のタイムスロットが表示され、空きのある時間帯を選択します
3. **メニューの選択：** 通常のメニュー画面と同じ操作でカートに追加します
4. **お客さん情報の入力：**
   - 名前（必須）
   - 電話番号（必須、10〜11桁、ハイフンなし）
5. **支払方法の選択：** 「店頭支払い」または「オンライン決済」
6. **注文の確定**

**バリデーション：**
- 品目数の上限（設定による、デフォルト10品）
- 金額の上限（設定による、デフォルト¥30,000）
- 受取時間が最低準備時間以降であること
- 選択した時間帯にスロット空きがあること

**注文確定後：**
- オンライン決済の場合：Stripe/Squareの決済ページにリダイレクトされます
- 店頭支払いの場合：注文確認画面が表示されます
- 注文IDと電話番号でステータスを確認できます
