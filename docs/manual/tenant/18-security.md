# 18. セキュリティ

POSLAを安全に利用するため、アカウント、パスワード、PIN、端末、権限管理の基本ルールを守ってください。

## 基本ルール

- 1人1アカウントで利用してください。
- 共有アカウントは使用しないでください。
- 退職者のアカウントは速やかに無効化してください。
- パスワードは他サービスと使い回さないでください。
- レジやKDSなどの店舗備品は端末専用アカウントで運用してください。
- 現金や返金に関わる操作では、担当スタッフのPINを使用してください。

## 権限管理

| ロール | 管理ポイント |
|---|---|
| オーナー | 全店舗に影響するため、必要な人だけに付与してください。 |
| マネージャー | 担当店舗の運用責任者に付与してください。 |
| スタッフ | 日常業務に必要な範囲で付与してください。 |
| 端末専用 | 店舗備品端末のみに使用してください。 |

## 監査ログ

POSLAでは、重要な操作が記録されます。返金、レジ操作、ユーザー変更、設定変更などに不審な点がある場合は、監査ログを確認してください。

## 端末管理

共有端末では、ブラウザの保存パスワードや個人アカウントの利用に注意してください。端末の紛失や盗難があった場合は、すぐにパスワード変更とアカウント無効化を行ってください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 不審なログインがある | パスワード変更、該当アカウントの無効化、POSLAサポートへの連絡を行ってください。 |
| PINが漏れた可能性がある | すぐにPINを変更してください。 |
| 退職者がログインできる | アカウントを無効化してください。 |
| 権限が強すぎる | ロールを見直してください。 |
