# 6. 注文管理

注文管理では、セルフオーダー、ハンディPOS、テイクアウトなどから入った注文を確認します。注文内容はKDS、レジ、レポートに連動します。

## 注文の入口

| 注文方法 | 主な利用場面 |
|---|---|
| セルフオーダー | お客様がテーブルQRから注文 |
| ハンディPOS | スタッフがお客様の代わりに注文入力 |
| テイクアウト | 店外・事前注文の受付 |
| 予約連動 | 予約来店後の着席・注文管理 |

## ハンディPOSで注文する

1. ハンディPOSを開きます。
2. 店舗とテーブルを確認します。
3. 商品を選び、数量とオプションを確認します。
4. 注文内容を送信します。
5. KDSに注文が表示されることを確認します。

## 注文履歴を確認する

注文履歴では、注文時刻、テーブル、商品、数量、ステータスを確認できます。問い合わせや会計確認の際に利用してください。

## 注文の変更・取消

注文送信後の変更は、厨房での調理状況や店舗ルールに合わせて行ってください。すでに調理済みの商品を取り消す場合は、会計・在庫・オペレーションへの影響を確認してください。

## ステータスの考え方

| ステータス | 意味 |
|---|---|
| 受付済み | 注文が登録された状態 |
| 調理中 | 厨房で対応中の状態 |
| 提供済み | 商品提供が完了した状態 |
| 会計済み | 会計が完了した状態 |

## 困ったとき

| 症状 | 対処 |
|---|---|
| KDSに注文が出ない | 店舗選択、ネットワーク、KDS画面の更新を確認してください。 |
| 注文内容が違う | 注文履歴とレシート内容を確認してください。 |
| 送信に失敗する | Wi-Fi接続を確認し、再送信してください。 |
| テーブルが見つからない | テーブル管理で登録状態を確認してください。 |
