# 12. シフト管理

シフト管理では、希望シフト、勤務表、出勤・退勤、ヘルプ要請、人件費の確認を行います。

## シフト作成の流れ

1. スタッフが希望シフトを提出します。
2. マネージャーが希望を確認します。
3. 必要人数、役割、休憩時間を調整します。
4. シフトを確定してスタッフに共有します。
5. 営業後に実績と差分を確認します。

## 勤怠打刻

スタッフは出勤時に出勤、退勤時に退勤を打刻します。GPS確認を利用している店舗では、店舗付近での打刻が必要です。

## ヘルプ要請

人員が不足する場合、他店舗へヘルプを依頼できます。依頼内容、日時、役割、必要人数を明確にしてください。

## 人件費確認

確定シフトと勤怠実績をもとに、人件費の目安を確認できます。給与計算の最終確定は、店舗または本部の運用ルールに従ってください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| 打刻できない | ログイン中のアカウント、位置情報許可、店舗付近かを確認してください。 |
| 希望シフトが見えない | 提出期間、対象スタッフ、表示週を確認してください。 |
| シフトが重複する | 同じスタッフの勤務日と時間帯を確認してください。 |
| 人件費が想定と違う | 時給設定、休憩、実績打刻を確認してください。 |
