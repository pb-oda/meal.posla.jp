# 3. 管理ダッシュボード

管理ダッシュボードは、店舗運営に必要な設定と確認を行う画面です。ログインしたユーザーのロールに応じて、表示されるタブが変わります。

## 画面の基本構成

| 場所 | 内容 |
|---|---|
| ヘッダー | 店舗名、店舗切替、ログイン中のユーザー名、パスワード変更 |
| メインタブ | メニュー、注文、レポート、店舗設定などの機能切替 |
| サブタブ | 各機能の中の詳細メニュー |
| 右下のAIヘルプデスク | 操作で困ったときに質問できるサポートボタン |

## ロール別の表示

| ロール | 主に表示される内容 |
|---|---|
| オーナー | 複数店舗管理、本部機能、契約・決済設定、ユーザー管理 |
| マネージャー | 担当店舗のメニュー、注文、会計、在庫、シフト、レポート、店舗設定 |
| スタッフ | 勤怠打刻、マイシフト、希望シフト提出 |

表示されないタブがある場合は、権限または所属店舗が不足している可能性があります。

## 店舗を切り替える

複数店舗に所属している場合は、画面上部の店舗セレクターから操作対象の店舗を選びます。選択した店舗に対して、メニュー、注文、レポートなどの操作が行われます。

## AIヘルプデスクを使う

管理画面右下のAIヘルプデスクから、操作方法やエラー時の対応を質問できます。質問するときは、画面名、操作内容、表示されたメッセージを一緒に入力すると回答が正確になります。

## 操作時の注意

- 複数店舗を管理している場合は、操作前に選択中の店舗名を確認してください。
- スタッフアカウントでは、店舗運営に必要な一部機能のみ表示されます。
- 画面表示が古い場合は、ブラウザの再読み込みを行ってください。

## 困ったとき

| 症状 | 対処 |
|---|---|
| タブが表示されない | 権限と所属店舗を確認してください。 |
| 店舗が選べない | オーナーまたはマネージャーに所属店舗の設定を確認してください。 |
| 読み込みに失敗する | 再読み込みし、改善しない場合はPOSLAサポートへお問い合わせください。 |
