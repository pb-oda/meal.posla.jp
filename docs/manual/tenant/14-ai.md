# 14. AIアシスタント

AIアシスタントは、店舗運営の補助として、投稿文作成、売上分析、競合調査、需要予測、メニュー質問対応などを行います。

## 主な機能

| 機能 | 内容 |
|---|---|
| SNS投稿作成 | メニューやキャンペーンに合わせた投稿文を作成 |
| 売上分析 | 売上傾向や改善のヒントを提示 |
| 競合調査 | 周辺店舗情報を参考に差別化案を提示 |
| 需要予測 | 過去傾向から仕込みや発注の参考値を提示 |
| AIウェイター | お客様からのメニュー質問に回答 |
| 予約AI解析 | 予約フォームの自由文から日時・人数を自動抽出 |
| AIキッチン最適化 | KDS の提供優先度を AI がアシスト |

## 利用時の注意

AIの回答は店舗運営を補助する参考情報です。価格、在庫、営業時間、アレルギー、予約可否など、お客様に影響する内容は必ず店舗側で最終確認してください。

::: tip 操作で困ったときは「サポートガイド」
管理画面右下の「📚 サポートガイド」ボタンから、FAQ、キーワード検索、症状別の案内、エラー番号検索を利用できます。操作方法を確認したいときは、まずサポートガイドをご利用ください。
:::

## 困ったとき

| 症状 | 対処 |
|---|---|
| AIが利用できない | 時間を置いて再試行し、続く場合はPOSLAサポートへお問い合わせください。 |
| 回答が店舗状況と違う | 店舗側の最新情報を優先してください。 |
| 音声入力が使えない | Google Chromeとマイク許可を確認してください。 |
| 予約日時の解釈が違う | 「2026年4月25日 19時」のように具体的に入力してください。 |
| 管理画面の操作で困った | 右下の「📚 サポートガイド」から FAQ / キーワード / 症状別に確認できます。 |
