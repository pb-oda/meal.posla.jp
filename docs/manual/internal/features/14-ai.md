# 14. AIアシスタント

管理ダッシュボードの「AIアシスタント」タブでは、AIを活用したSNS投稿生成・売上分析・競合調査・需要予測の機能を利用できます。マネージャー以上のロールで利用できます。

AIにはGemini 2.0 Flash APIが使用されており、APIキーはPOSLA運営が一括管理しているため、テナント側での設定は不要です。

## 14.1 SNS投稿生成

InstagramやX（旧Twitter）向けの投稿文を自動生成します。

### 入力フィールド

| フィールド | 説明 |
|-----------|------|
| 店舗名 | 投稿に含める店舗名 |
| おすすめメニュー | 紹介するメニュー品目 |
| アピールポイント | 強調したいポイント |

### プラットフォーム別の生成

| プラットフォーム | 特徴 |
|---------------|------|
| Instagram | 絵文字使用、ハッシュタグ5つ以上、複数行の読みやすい形式 |
| X（Twitter） | テキストのみ、簡潔な文字数に収まる形式 |

### AIパラメータ

| パラメータ | 値 |
|-----------|-----|
| モデル | Gemini 2.0 Flash |
| Temperature | 0.8（クリエイティブ） |
| 最大出力トークン | 1024 |

---

## 14.2 売上分析

店舗の売上データをAIが分析し、トレンドや改善点を提案します。

### 分析の流れ

1. AIアシスタントの「売上分析」を選択します
2. 売上レポートAPIからデータが自動取得されます
   - トップ10品目
   - 時間帯別売上
   - サービス時間データ
3. AIが分析結果を返します

### AIからの出力

- 売上トレンドの分析
- ベストセラー品目の特定
- 改善が必要な領域の指摘
- 具体的な改善提案

### AIパラメータ

| パラメータ | 値 |
|-----------|-----|
| Temperature | 0.5（分析向け、やや低め） |
| 最大出力トークン | 1024 |

---

## 14.3 競合調査

周辺の競合店舗をGoogle Maps/Places APIで分析し、差別化戦略を提案します。

### 分析内容

- 近隣店舗の価格帯
- 評価（レーティング）
- 推定客数
- 競合との差別化ポイント

### AIからの出力

- 競合店舗名と数値データを含む分析
- 今週中に実行可能な3つの具体的アクション
- 店舗名や数値を含む具体的な提案

---

## 14.4 需要予測

過去の売上データとレシピ情報を組み合わせて、今後の需要を予測します。

### データソース

- 過去7日間の売上データ
- 品目別の販売数量
- 曜日別パターン
- レシピ情報（原材料の使用量）

### AIからの出力

| セクション | 内容 |
|-----------|------|
| 日別需要予測 | 今後の日ごとの品目別販売予測 |
| 原材料発注推奨 | 不足見込みの原材料と推奨発注量 |
| 低在庫アラート | 閾値を下回る見込みの原材料 |
| ポーション調整 | 使い切り・廃棄削減のための提案 |

### AIパラメータ

| パラメータ | 値 |
|-----------|-----|
| Temperature | 0.5（分析向け） |
| 最大出力トークン | 8192（詳細な予測のため大きめ） |

---

## 14.5 AIウェイター（お客さん向け）

セルフオーダー画面には、お客さんが利用できるAIチャットボット「AIウェイター」が搭載されています。

### 表示

- 画面右下にオレンジ色のフローティングボタン（FAB）として表示されます
- タップするとチャットパネル（320px × 420px）が開きます

### 機能

| 機能 | 説明 |
|------|------|
| メニュー相談 | 「辛くないおすすめは？」「子供向けメニューは？」などの質問に回答 |
| おすすめ提案 | AIがメニューからおすすめ品目をカード形式で提案 |
| カートに追加 | 提案されたメニューを直接カートに追加 |
| 音声入力 | Web Speech APIによる音声入力（日本語） |
| 会話履歴 | 最大10往復の会話を保持 |

### チャットパネルの構成

| 要素 | 説明 |
|------|------|
| ウェルカムメッセージ | 最初に表示される案内と質問例 |
| メッセージ履歴 | ユーザーとAIのチャット履歴 |
| タイピングインジケーター | AI応答中のアニメーション |
| おすすめカード | 品名、価格、「カートに追加」ボタン |
| テキスト入力欄 | — |
| 音声入力ボタン | マイクアイコン |

::: info AIウェイターの利用条件
AIウェイターは、POSLA運営でGemini APIキーが設定されている場合に自動的に有効になります。APIの利用可否はシステムが自動チェックし、利用できない場合はボタンが表示されません。
:::
